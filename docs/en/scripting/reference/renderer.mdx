# Renderer Proxy

**Global Binding:** `renderer`

The renderer provides a complete 2D drawing API drawn through the **divinity-backed canvas**. All drawing calls must be made inside a `renderScreen` event callback, where the canvas frame is active.

<Callout type="info">
    All color parameters expect a packed **ARGB Integer**. Create these using `renderer.color(r, g, b, a)` or the `Color` Java class.
</Callout>

## Images

Images are loaded once into a handle and drawn by that handle. Loaded images are cached by the renderer, so calling `loadImage` repeatedly with the same path is cheap.

<Callout type="warning">
    The old raw-integer `image(handle, ...)` form has been **removed** — there was never a way to obtain a valid raw id, so it never worked. Load images with `loadImage` and pass the returned `ImageHandle` instead.
</Callout>

<Method name="loadImage" signature="ImageHandle loadImage(String path)" description="Loads an image and returns a handle to it. The handle is cached by the renderer. Returns ImageHandle.INVALID on failure — check validity before drawing." params={[{ name: "path", type: "String", desc: "Resource path to the image." }]} returns={{ type: "ImageHandle", desc: "Handle to the loaded image, or ImageHandle.INVALID on failure." }} />

<Method name="image" signature="void image(ImageHandle handle, double x, double y, double width, double height, double radius)" description="Draws a previously loaded image at the given bounds with optional corner rounding." params={[{ name: "handle", type: "ImageHandle", desc: "Handle returned by loadImage." }, { name: "x", type: "double", desc: "X coordinate." }, { name: "y", type: "double", desc: "Y coordinate." }, { name: "width", type: "double", desc: "Destination width." }, { name: "height", type: "double", desc: "Destination height." }, { name: "radius", type: "double", desc: "Corner radius applied to the image." }]} />

<Method name="imageTinted" signature="void imageTinted(ImageHandle handle, double x, double y, double width, double height, double radius, double tint)" description="Draws a loaded image multiplied by a tint color — useful for recoloring icons or fading images." params={[{ name: "handle", type: "ImageHandle", desc: "Handle returned by loadImage." }, { name: "x", type: "double", desc: "X coordinate." }, { name: "y", type: "double", desc: "Y coordinate." }, { name: "width", type: "double", desc: "Destination width." }, { name: "height", type: "double", desc: "Destination height." }, { name: "radius", type: "double", desc: "Corner radius applied to the image." }, { name: "tint", type: "double", desc: "ARGB tint color multiplied over the image." }]} />

<Method name="destroyImage" signature="void destroyImage(ImageHandle handle)" description="Releases a previously loaded image handle." params={[{ name: "handle", type: "ImageHandle", desc: "Handle returned by loadImage." }]} />

<Callout type="info">
    `image` and `imageTinted` work inside the `renderScreen` callback. Drawing images from the GPU/extract render path is not supported from scripts.
</Callout>

```javascript
let icon = null;

module.on("enable", () => {
    icon = renderer.loadImage("opal:textures/logo.png");
});

module.on("renderScreen", (event) => {
    if (icon == null || !icon.isValid()) return;
    renderer.image(icon, 10, 10, 32, 32, 4);
    renderer.imageTinted(icon, 50, 10, 32, 32, 4, renderer.color(255, 120, 120));
});

module.on("disable", () => {
    if (icon != null) renderer.destroyImage(icon);
});
```

```javascript
// Draw a triangle
renderer.beginPath();
renderer.moveTo(100, 100);
renderer.lineTo(150, 50);
renderer.lineTo(200, 100);
renderer.closePath();

renderer.strokeWidth(2);
renderer.stroke();

// Draw a curve

renderer.moveTo(100, 150);
renderer.quadTo(150, 100, 200, 150);
renderer.cubicTo(220, 170, 260, 130, 300, 150);
renderer.strokeColor(0xFFFFFFFF);
renderer.strokeWidth(2);
renderer.stroke();
```

## Full Example

```javascript
module.on("renderScreen", (event) => {
    const sw = client.getScaledWidth();
    const panelWidth = 140;
    const x = sw - panelWidth - 10;
    const y = 10;

    // Soft shadow under the panel, then a frosted-glass background
    renderer.shadow(x, y, panelWidth, 60, 6, 12, 0, 3, renderer.color(0, 0, 0, 130));
    renderer.blurFill(x, y, panelWidth, 60, 6);
    renderer.roundedRect(x, y, panelWidth, 60, 6, renderer.color(15, 15, 20, 160));

    // Title with theme color
    renderer.textShadow("productsans-bold", "Status", x + 10, y + 16, 9, client.getThemePrimary());

    // Stats
    renderer.text("productsans-medium", "FPS: " + client.getFPS(), x + 10, y + 34, 7, renderer.color(180, 180, 180));
    renderer.text("productsans-medium", "BPS: " + movement.getBlocksPerSecond().toFixed(1), x + 10, y + 48, 7, renderer.color(180, 180, 180));
});
```
