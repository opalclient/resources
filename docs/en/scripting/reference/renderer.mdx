# Renderer Proxy

A complete 2D drawing API backed by the active **divinity canvas**, exposed to scripts as the global `renderer` binding.

---

## Renderer Proxy
**Global Binding:** `renderer`

Draws filled shapes, gradients, outlines, composite effects, images, vector paths, and text onto the divinity-backed canvas using the Minecraft scaled coordinate system.

<Callout type="warning">
    Every draw call must run **inside an active canvas frame** — that means from a module's `renderScreen` callback, from a command-palette view's `render`, or from a Dynamic Island `render` callback (see the Overlay Proxy). Outside an active frame the canvas is unavailable and image draws (which need a live extract context) are silently skipped.
</Callout>

<Callout type="info">
    **Numbers and colors.** JavaScript has a single `number` type (an IEEE‑754 `double`), so every parameter here is declared `double` and cast internally — GraalVM will not implicitly narrow. Colors are packed **ARGB integers**. Build them with `renderer.color(r, g, b[, a])` rather than raw `0xAARRGGBB` literals: a hex literal whose alpha byte is `0x80`–`0xFF` is greater than 2³¹ as a JS `double` and narrows to the wrong `int`, so `0xFFFFFFFF` does **not** round‑trip. `renderer.color(255, 255, 255)` does.
</Callout>

<Callout type="info">
    `image` and `imageTinted` route through the active extract context. When no render pass is open the draw is silently skipped — call them from within `renderScreen`, a palette view's `render`, or an island `render`.
</Callout>

```javascript
let icon = null;

module.on("enable", () => {
    icon = renderer.loadImage("opal:textures/logo.png");
});

module.on("renderScreen", (event) => {
    if (icon == null || !icon.isValid()) return;
    renderer.image(icon, 10, 10, 32, 32, 4);
    renderer.imageTinted(icon, 50, 10, 32, 32, 4, renderer.color(255, 120, 120));
});

module.on("disable", () => {
    if (icon != null) renderer.destroyImage(icon);
});
```

renderer.strokeWidth(2);
renderer.stroke();

// Draw a curve

renderer.moveTo(100, 150);
renderer.quadTo(150, 100, 200, 150);

renderer.strokeColor(renderer.color(255, 255, 255));

## Full Example

```javascript
module.on("renderScreen", (event) => {
    const sw = client.getScaledWidth();
    const panelWidth = 140;
    const x = sw - panelWidth - 10;
    const y = 10;

    // Soft shadow under the panel, then a frosted-glass background
    renderer.shadow(x, y, panelWidth, 60, 6, 12, 0, 3, renderer.color(0, 0, 0, 130));
    renderer.blurFill(x, y, panelWidth, 60, 6);
    renderer.roundedRect(x, y, panelWidth, 60, 6, renderer.color(15, 15, 20, 160));

    // Title with theme color
    renderer.textShadow("productsans-bold", "Status", x + 10, y + 16, 9, client.getThemePrimary());

    // Stats
    renderer.text("productsans-medium", "FPS: " + client.getFPS(), x + 10, y + 34, 7, renderer.color(180, 180, 180));
    renderer.text("productsans-medium", "BPS: " + movement.getBlocksPerSecond().toFixed(1), x + 10, y + 48, 7, renderer.color(180, 180, 180));
});
```
