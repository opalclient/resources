## World Proxy

<Callout type="info">
    There is no `getBlockState` or `getBlock`. A raw block state carries nothing a script could read, so the queries above are flattened onto `world` directly: `isAir`, `isSolid`, `isReplaceable`, `getBlockName`, and `getBlockHardness` cover what those methods were used for.
</Callout>

### Block Helpers

<Method name="hasAdjacentBlock" signature="boolean hasAdjacentBlock(BlockPos pos)" description="Whether any face-adjacent block is solid (placeable-against)." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "boolean", desc: "True if has neighbor." }} />
<Method name="getAdjacentDirections" signature="List&lt;Direction&gt; getAdjacentDirections(BlockPos pos)" description="Returns directions where a solid neighbour exists. Can be empty. A List wrapper, not a JS array: walk it with size() and get(i)." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "List<Direction>", desc: "Directions with a solid neighbour." }} />

```javascript
const below = new BlockPos(pos.getX(), pos.getY() - 1, pos.getZ());

if (!world.isReplaceable(below)) return;

const dirs = world.getAdjacentDirections(below);
if (dirs.isEmpty()) return;

const side = dirs.get(0);
const neighbor = below.offset(side);
```

### Entity Queries

<Method name="getEntities" signature="List&lt;Entity&gt; getEntities()" description="Returns all entities in the world. A List wrapper, not a JS array and not iterable." returns={{ type: "List<Entity>", desc: "All entities." }} />
<Method name="getLivingEntitiesInRange" signature="List&lt;Entity&gt; getLivingEntitiesInRange(double radius)" description="Returns living entities (mobs + players) within radius of the player, excluding self. A List wrapper, not a JS array." params={[{ name: "radius", type: "double", desc: "Search radius in blocks." }]} returns={{ type: "List<Entity>", desc: "Nearby living entities." }} />

<Callout type="warning">
    Both return a [`List`](types#list) wrapper. `for...of`, `.length`, and `[i]` indexing **do not work on it**, and they fail silently rather than throwing. Walk it with `size()` and `get(i)`.
</Callout>

```javascript
// Attack closest entity
const targets = world.getLivingEntitiesInRange(5.0);
if (!targets.isEmpty()) {
    mc.interactionManager.attackEntity(targets.get(0));
    player.swingHand(MAIN_HAND);
}

// Walk every entity in the world
const all = world.getEntities();
for (let i = 0; i < all.size(); i++) {
    const entity = all.get(i);
    if (entity.isPlayer()) client.print(entity.getName());
}
```
