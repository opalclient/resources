### Block Helpers

<Method name="hasAdjacentBlock" signature="boolean hasAdjacentBlock(BlockPos pos)" description="Whether any face-adjacent block is solid (placeable-against)." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "boolean", desc: "True if has neighbor." }} />
<Method name="getAdjacentDirections" signature="List&lt;Direction&gt; getAdjacentDirections(BlockPos pos)" description="Returns directions where a solid neighbour exists. Can be empty." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "List<Direction>", desc: "List of ScriptDirection." }} />

```javascript
const below = new BlockPos(pos.getX(), pos.getY() - 1, pos.getZ());

if (!world.isReplaceable(below)) return;

const dirs = world.getAdjacentDirections(below);
if (dirs.isEmpty()) return;

const side = dirs.get(0);
const neighbor = below.offset(side);
```

### Entity Queries

<Method name="getEntities" signature="Iterable getEntities()" description="Returns all entities in the world." returns={{ type: "Iterable", desc: "All entities." }} />
<Method name="getLivingEntitiesInRange" signature="List&lt;LivingEntity&gt; getLivingEntitiesInRange(double radius)" description="Returns living entities (mobs + players) within radius of the player, excluding self." params={[{ name: "radius", type: "double", desc: "Search radius in blocks." }]} returns={{ type: "List<LivingEntity>", desc: "Nearby living entities." }} />

```javascript
// Attack closest entity
const targets = world.getLivingEntitiesInRange(5.0);
if (!targets.isEmpty()) {
    mc.interactionManager.attackEntity(targets.get(0));
    player.swingHand(MAIN_HAND);
}
```
