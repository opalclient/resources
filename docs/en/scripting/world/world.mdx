# World Proxy

Block queries, entity queries, and world info.

---

## World Proxy
**Global Binding:** `world`

### Block Queries

<Method name="isReplaceable" signature="boolean isReplaceable(BlockPos pos)" description="Whether the block can be replaced (air, fluid, grass, etc.)." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "boolean", desc: "True if replaceable." }} />
<Method name="isSolid" signature="boolean isSolid(BlockPos pos)" description="Whether the block is solid." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "boolean", desc: "True if solid." }} />
<Method name="getBlockName" signature="String getBlockName(BlockPos pos)" description="Returns the localized display name of the block. <Method name="getBlockId" signature="String getBlockId(BlockPos pos)" description="Returns the block's registry id, independent of the client's language. The locale-safe way to match a specific block." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "String", desc: "Registry id, e.g. 'minecraft:stone'. Never null." }} />
<Method name="getBlockHardness" signature="float getBlockHardness(BlockPos pos)" description="Returns the breaking hardness. -1 means unbreakable (bedrock)." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "float", desc: "Hardness." }} />

<Callout type="info">
    There is no `getBlockState` or `getBlock`. A raw block state carries nothing a script could read, so the queries above are flattened onto `world` directly: `isAir`, `isSolid`, `isReplaceable`, `getBlockName`, `getBlockId`, and `getBlockHardness` cover what those methods were used for.
</Callout>

<Callout type="info">
    Prefer `getBlockId` for matching (`if (world.getBlockId(pos) === "minecraft:stone")`) and `getBlockName` for text shown to the player. An unloaded position resolves both to air, mirroring each other rather than throwing or returning null: `getBlockId` returns `"minecraft:air"`, and `getBlockName` returns air's localized name.
</Callout>

### Block Helpers

<Method name="hasAdjacentBlock" signature="boolean hasAdjacentBlock(BlockPos pos)" description="Whether any face-adjacent block is solid (placeable-against)." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "boolean", desc: "True if has neighbor." }} />
<Method name="getAdjacentDirections" signature="List&lt;Direction&gt; getAdjacentDirections(BlockPos pos)" description="Returns directions where a solid neighbour exists. Can be empty. A read-only ScriptList: iterate it with for..of or index it, and size()/get(i) also work." params={[{ name: "pos", type: "BlockPos", desc: "Block position." }]} returns={{ type: "List<Direction>", desc: "Directions with a solid neighbour." }} />

```javascript
const below = new BlockPos(pos.getX(), pos.getY() - 1, pos.getZ());

if (!world.isReplaceable(below)) return;

const dirs = world.getAdjacentDirections(below);
if (dirs.isEmpty()) return;

const side = dirs.get(0);
const neighbor = below.offset(side);
```

### Entity Queries

<Method name="getEntities" signature="List&lt;Entity&gt; getEntities()" description="Returns all entities in the world. A read-only ScriptList: iterate it with for..of or index it, and size()/get(i) also work." returns={{ type: "List<Entity>", desc: "All entities." }} />
<Method name="getLivingEntitiesInRange" signature="List&lt;Entity&gt; getLivingEntitiesInRange(double radius)" description="Returns living entities (mobs + players) within radius of the player, excluding self. A read-only ScriptList." params={[{ name: "radius", type: "double", desc: "Search radius in blocks." }]} returns={{ type: "List<Entity>", desc: "Nearby living entities." }} />

<Callout type="info">
    Both return a read-only [`ScriptList`](types#list). It reads as a real array: `for..of`, `.length`, `[i]` indexing, and spread all work, and so do `size()`/`get(i)`. The write and `Array.prototype` limits are in [Types](types#list).
</Callout>

```javascript
// Attack closest entity
const targets = world.getLivingEntitiesInRange(5.0);
if (!targets.isEmpty()) {
    mc.interactionManager.attackEntity(targets.get(0));
    player.swingHand(MAIN_HAND);
}

// Walk every entity in the world
for (const entity of world.getEntities()) {
    if (entity.isPlayer()) client.print(entity.getName());
}
```
