# Modules Proxy

**Global Binding:** `modules`

Dedicated access to the client's module system: querying state, toggling native modules, reading metadata, and listing modules by category.

All lookups are **case-insensitive**, and module names are matched after spaces are normalized to underscores (so `"Kill Aura"` and `"killaura"` resolve the same target).

<Callout type="info">
    Every method returns a safe default if the requested module does not exist: `false` for booleans, `null` for strings, and an empty list for listings. Lookups never throw into the script runtime.
</Callout>

## Listing

<Callout type="warning">
    Listing methods return a [`List`](../world/types#list) wrapper, **not a JavaScript array**. `.length`, `[i]` indexing, `for...of`, and `Array.from()` do not work on it, and they do not throw either: they evaluate to `undefined` or an empty array, so a loop over `.length` runs zero times and your script silently does nothing. Walk it with `size()` and `get(i)`.
</Callout>

<Method
    name="listAll"
    signature="List&lt;String&gt; listAll()"
    description="Returns the display names of all registered modules, both native and script-defined."
    returns={{ type: "List<String>", desc: "Module names. A List wrapper: use size() and get(i)." }}
/>

<Method

## Example

```javascript
script.registerModule({
    name: "AutoDisableFlight",
    description: "Turns off Flight whenever KillAura starts"
}, (module) => {
    module.on("preGameTick", () => {
        // Read native module state.
        if (modules.isEnabled("KillAura") && modules.isEnabled("Flight")) {
            modules.setEnabled("Flight", false);
            client.print("Disabled Flight while KillAura is active");
        }

        // Walk every Combat module and report its state.
        // size()/get(i), never .length/[i] — the list is not a JS array.
        const combat = modules.listCategory("Combat");
        for (let i = 0; i < combat.size(); i++) {
            const name = combat.get(i);
            client.print(name + " -> " + modules.isEnabled(name));
        }
    });
});
```
