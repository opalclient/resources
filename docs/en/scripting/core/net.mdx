# Net Proxy

**Global Binding:** `net`

Curated serverbound packet sends. Rather than going through a higher-level proxy method, a script builds and sends a handful of real vanilla packets directly.

<Callout type="info">
    Every send here is a genuine packet over the network connection (a no-op with no connection), subject to the same [`sendPacket` / `instantaneousSendPacket`](../events#packet-events) events as any other outbound packet.
</Callout>

<Callout type="warning">
    **Invalid input throws.** An unrecognized action/mode string, or an out-of-range numeric parameter, throws a catchable error whose message lists the valid values â€” wrap the call in `try`/`catch` if the input isn't already validated by your own logic.
</Callout>

## Player Command Actions

Valid `action` strings for `playerCommand`:

| Action |
| :--- |
| `stopSleeping` |
| `startSprinting` |
| `stopSprinting` |
| `startRidingJump` |
| `stopRidingJump` |
| `openInventory` |
| `startFallFlying` |

## Player Action Actions

Valid `action` strings for `playerAction`. **Dig actions require the 3-argument overload** (`blockPos` + `face`); position-free actions work with either overload.

| Action | Kind |
| :--- | :--- |
| `startDestroyBlock` | Dig â€” requires `blockPos` + `face` |
| `stopDestroyBlock` | Dig â€” requires `blockPos` + `face` |
| `abortDestroyBlock` | Dig â€” requires `blockPos` + `face` |
| `dropAllItems` | Position-free |
| `dropItem` | Position-free |
| `releaseUseItem` | Position-free |
| `swapItemWithOffhand` | Position-free |

## Slot Click Modes

Valid `mode` strings for `slotClick`. This is the **same camelCase token set** [`preSlotClick`'s `getMode()`](../events#player-interaction-events) reports back, symmetrically.

| Mode |
| :--- |
| `pickup` |
| `quickMove` |
| `swap` |
| `clone` |
| `throw` |
| `quickCraft` |
| `pickupAll` |

## Example

```javascript
script.registerModule({
    name: "NetDemo",
    description: "Exercises the net proxy"
}, (module) => {
    module.on("preGameTick", () => {
        if (mc.getPlayer() === null || mc.getWorld() === null) return;

        // Ghost-swing the off hand â€” the server sees it, nothing plays locally.
        net.swing(OFF_HAND);
    });

    module.on("keyPress", (event) => {
        if (event.getCode() !== keys.F6) return;

        // Flick to hotbar slot 3 with a raw packet.
        net.heldSlot(3);

        // Position-free player action.
        net.playerAction("swapItemWithOffhand");

        // Dig action requires blockPos + face.
        const pos = player.getBlockPosition();
        const dirs = world.getAdjacentDirections(pos);
        if (!dirs.isEmpty()) {
            net.playerAction("startDestroyBlock", pos, dirs.get(0));
        }

        // Quick-move whatever is in container slot 0 (no-op if nothing is open).
        try {
            net.slotClick(0, 0, "quickMove");
        } catch (e) {
            client.error("slotClick failed: " + e.message);
        }
    });
});
```

<Callout type="info">
    `net.slotClick` is gated by the cancellable [`preSlotClick`](../events#player-interaction-events) event â€” a script (or another module) that cancels it blocks the click, the same as it would for a real inventory click. See [Events](../events#cancelling-events) for the full cancellation model.
</Callout>
