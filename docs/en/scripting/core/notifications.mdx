# Notification Proxy

Temporary toast notifications on the HUD.

---

## Notification Proxy
**Global Binding:** `notification`

Displays temporary "toast" notifications on the HUD.

<Callout type="info">
    Toast notifications (this page) and [Dynamic Island overlays](overlay) both render through the client's shared overlay kit internally, but they are separate proxies with separate global bindings. Calling `notification.*` never touches an island, and calling `overlay.*` never shows a toast.
</Callout>
