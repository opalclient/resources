# Interaction Manager

Block interaction, entity attacks, and block breaking, accessed through `mc.interactionManager`.

---

## InteractionManager
**Access via:** `mc.interactionManager`

Wraps `ClientPlayerInteractionManager`. The local player is injected automatically, and every method is null-guarded internally, so it is safe to call before the world or player are ready.

### Example: Scaffold Pattern

```javascript
const below = new BlockPos(
    player.getBlockPosition().getX(),
    player.getBlockPosition().getY() - 1,
    player.getBlockPosition().getZ()
);

if (!world.isReplaceable(below)) return;

const dirs = world.getAdjacentDirections(below);
if (dirs.isEmpty()) return;

const side = dirs.get(0);
const neighbor = below.offset(side);

const raytrace = rotation.getRotationFromRaycastedBlock(
    neighbor,
    side.getOpposite(),
    rotation.getRotation(),
    player.getEyePosition()
);
if (raytrace === null) return;

rotation.set(raytrace.getYaw(), raytrace.getPitch());
mc.interactionManager.interactBlock(MAIN_HAND, raytrace.getHitResult());
player.swingHand(MAIN_HAND);
```
