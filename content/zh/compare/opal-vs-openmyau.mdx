## 免费版与$8.99：每个价格真正买到的东西

OpenMyau 成本为零，这并非脚注，而是重点：它采用开源模式，因此您可以阅读代码、自行构建，而无需触及任何您无法控制的启动器或账户系统。这是运行它的一个真正理由，也是它能够作为 Opai 内部的 Myau 扩展存在，而不是需要自己的商店的原因。

Opal 成本为 $8.99。购买的是别人已经为您完成的客户端：一个可以登录您的启动器，一个自定义玻璃 UI 引擎（而不是原始设置屏幕），以及 70 个涵盖战斗、移动、视觉、世界和实用工具的模块，这些模块已经连接好，而不是让您自行组装。您并非为 OpenMyau 自己的列表声称所缺乏的东西付费。您是在为跳过构建步骤而付费。

## 两种截然不同的 Minecraft 版本

OpenMyau 针对 MC 1.8.9，许多竞技 Hypixel PvP 仍然运行的版本，这并非贬低。1.8.9 的战斗仍然是许多玩家想要的标配，而专门为此构建的开源客户端显然拥有一个专注的受众.

Opal 运行在当前版本，MC 26.2，作为 Fabric 模组在 Java 25 上。如果您的服务器已经升级到 1.8.9 以上的版本，或者您只是想要一个针对最新客户端内部结构构建的客户端，那么 Opal 是唯一一个实际运行在这个版本上的。这两个版本并非真正竞争同一个安装。它们运行在完全不同的 Minecraft 版本上
## 实际记录了什么，以及什么没有记录
Opal 的列表详细到模块级别：共有 70 个，分别分布在战斗、移动、视觉、世界和工具模块中，外加一个嵌入式的 GraalJS 脚本运行时，用于编写针对客户端的自定义脚本，支持 Discord 丰富的游戏内显示，客户端内的 IRC 频道，以及动态的岛屿 HUD 覆盖层。

相比之下，OpenMyau 的列表则较为简略：仅有一行描述，称其为开源的 Hypixel 作弊客户端，以及它现在作为 Opai 的 Myau 扩展一并发布的这一事实。与 Opal 相比，它没有公开模块数量、功能列表或 UI 描述。这并不一定意味着它有什么缺点，开源意味着任何人都可以直接阅读源代码，而不是相信列表中的描述。这确实意味着本次比较只能说明实际记录的内容，而不能说明功能是否对等。自行阅读源代码是一个真正好的选择。你是否愿意在第一局游戏前花一个晚上来做这件事，是另一个问题
## 每个人实际上是为谁而存在的

这些产品服务于不同的用户，而不是直接竞争。如果你想要免费的，想要自己阅读代码，或者你已经在 MC 1.8.9 上并且对现状感到满意，OpenMyau 是一个合法的、可信的选择，免费且开源是 Opal 无法比拟的优势。如果你想要一个在最新 Minecraft 版本上维护的客户端，拥有完善的 UI 并且在登录时就提供广泛的模块，而 $8.99 不是问题，那么 Opal 就是为你打造的。
