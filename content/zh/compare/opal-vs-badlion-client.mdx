## 你实际安装的是什么

Badlion Client 是一个独立的应用程序。它不是一个你放入现有 Fabric 或 Forge 设置中的模组，因此你无需追踪或匹配你的 Minecraft 安装的加载器版本。如果你宁愿永远不用再考虑模组加载器，这确实是它的一个优点：Badlion 拥有端到端的完整流程，并且由于其独立性，它不受 Fabric 发布节奏或随之而来的兼容性破坏的影响。

Opal 实际上是一个 Fabric 模组，但它通过自己的启动器发布。你无需手动安装 Fabric Loader 或寻找匹配的 Fabric API 构建版本；启动器会为你处理这些底层工作。实际上，两者的设置难度都接近于零。在架构上，它们以两种不同的方式解决了相同的问题，如果你有一个“永远不需要模组加载器”的硬性要求，那么 Badlion 的模式能更直接地满足你的需求。
