### 幽灵与外露，一应俱全

大多数客户端会让你做出选择。幽灵客户端提供微妙的功能，但没有绕过机制。外露客户端提供绕过机制，但没有幽灵模式。Opal 两者兼备，让你能够根据情况选择开启潜行瞄准辅助或全杀光环。

### 平台，而非仅仅一个 JAR 文件

其他客户端只会给你一个文件和一个 Discord 邀请。Opal 提供了更多：

- 一个桌面启动器，负责安装、登录和更新。
- 一个网页仪表盘，用于管理你的账户、HWID 重置、下载和更新日志。
- 一个配置浏览器，用于查找和分享模块预设。
- 社区论坛，在这里你可以提交 bug 报告和功能请求，影响开发路线图。
- 一个 Discord 机器人，用于许可证管理和技术支持。

### GraalVM 脚本引擎

Opal 包含完整的 JavaScript 运行时。你可以编写脚本，与客户端的 API 对话，自动化任务，并构建你自己的模块。它使用 GraalVM，与企业级 Java 相同的引擎，在 Minecraft 中运行。这并非宏系统。 详情请参见 [脚本引擎介绍](/zh/blog/introducing-opal-scripting-engine)。

### 跨平台

Windows、Intel 和 Apple Silicon 上的 macOS，以及 Linux。你无需选择特定的操作系统。
