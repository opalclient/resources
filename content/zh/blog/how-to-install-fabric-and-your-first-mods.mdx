安装模组比看起来容易。你安装 [Fabric](https://fabricmc.net) 启动器，添加 [Fabric API](https://modrinth.com/mod/fabric-api)（大多数模组需要的库），然后将几个模组 jar 文件放入一个文件夹中。一个不错的入门组合是 [Sodium](https://modrinth.com/mod/sodium) 和 [ModMenu](https://modrinth.com/mod/modmenu)。 这就是从原版游戏到可用的模组游戏的过程，为客户端做好准备.

<KeyTakeaways>

- 首先安装 Fabric Loader，然后安装 Fabric API。
- 为了提升 FPS，使用 Sodium；ModMenu 用于在游戏中管理你的模组。
- 像 Opal 这样的客户端安装到同一个模组文件夹中。
- 如果你计划使用 Opal，请不要安装 Lithium；它们不兼容.

</KeyTakeaways>

## 逐步安装

<Steps>

<Step number={1} title="安装 Fabric 载入器">运行适用于你的 Minecraft 版本的 [Fabric](https://fabricmc.net) 安装程序。它会添加一个新的启动器配置文件，你可以在启动前选择</Step>

<Step number={2} title="添加Fabric API">[Fabric API](https://modrinth.com/mod/fabric-api) 是几乎所有 Fabric 模组所依赖的共享库。下载与您的版本相符的版本，并将其放入 `mods` 文件夹中。没有它，大多数模组将无法加载</Step>

<Step number={3} title="添加你的第一个模组">将 [Sodium](https://modrinth.com/mod/sodium) (帧数) 和 [ModMenu](https://modrinth.com/mod/modmenu) (游戏内管理和配置模组) 放入 `mods` 文件夹。 如果需要着色器，请添加 [Iris](https://modrinth.com/mod/iris)</Step>

<Step number={4} title="启动并检查">在 Fabric 档案中启动游戏，打开 ModMenu，并确认你的模组已列出。如果缺少任何内容，请检查其版本是否与你的 Minecraft 匹配，以及 Fabric API 是否已安装</Step>

</Steps>

<Callout variant="info" title="关于锂的说明">
许多入门指南会建议将锂作为第三个模组添加。它本身是一个不错的游戏逻辑优化器，但与Opal不兼容。如果您计划运行Opal，请使用Sodium来提升性能，从一开始就不要添加锂。
</Callout>

## 添加客户端

一旦加载器、Fabric API 和你的第一个模组都已安装，文件夹就可以准备客户端了。Opal 是一个针对 Minecraft 1.21.10 的 Fabric 模组，因此它的安装方式与其他模组相同：放入 `mods` 文件夹中。你可以保留你的性能优化设置，并添加 200 多个战斗、移动、视觉和实用工具方面的特性，一个玻璃 UI，以及一个脚本引擎。保留 Sodium 以提供性能空间，并移除 Lithium，你就可以开始了。
