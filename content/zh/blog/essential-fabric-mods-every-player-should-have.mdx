一些模组涵盖了良好 [Fabric](https://fabricmc.net) 设置的基础：以 [Fabric API](https://modrinth.com/mod/fabric-api) 作为基础，[Sodium](https://modrinth.com/mod/sodium) 用于性能提升，[Iris](https://modrinth.com/mod/iris) 用于着色器，以及 [ModMenu](https://modrinth.com/mod/modmenu) 用于管理所有内容。首先安装这些模组，大多数其他模组就能顺利安装在它们之上，包括客户端。对于 Opal 用户请注意：Lithium 是一个常见的选择，但它与 Opal 不兼容，所以应该将其排除.

<KeyTakeaways>

- Fabric API 是基础。在安装其他任何东西之前先安装它。
- Sodium 用于提升帧数，Iris 用于着色器，ModMenu 用于管理所有内容。
- Lithium 单独使用效果很好，但与 Opal 不兼容。
- Opal 是客户端层；它安装在与其它文件相同的文件夹中.

</KeyTakeaways>

## Fabric API

几乎所有 Fabric 模组都依赖的共享库。请先安装它。没有它，大多数模组将无法加载。与其说是功能，不如说是必需品。

## Sodium

重写渲染引擎，可提高 FPS，对性能较弱的 GPU 压力最大。这是你能获得的最大的性能提升，并且可以与 Opal 一起正常运行。

## Iris

运行着色器包，并且设计用于与 Sodium 配合使用，因此你可以在不降低帧数的情况下获得精美的视觉效果。如果你不关心着色器，可以跳过它。

## ModMenu

添加游戏内的模组列表和配置屏幕，让你无需编辑文件即可查看已安装的模组并更改设置。提升生活品质，让其他模组更容易使用。

## 关于 Lithium 的说明

[Lithium](https://modrinth.com/mod/lithium) 优化游戏逻辑，并且在原版增强设置下表现出色。不过，它与 Opal 不兼容，因此如果你的文件夹中包含 Opal，请不要使用 Lithium，而是依赖 Sodium 来提高性能。
