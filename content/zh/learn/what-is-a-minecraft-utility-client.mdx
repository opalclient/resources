## 实用客户端的作用

一个实用客户端将有用的功能打包到一个 mod 中，只需安装一次。 大部分 mod 旨在满足相同的一组需求：更简洁的界面、更好的性能以及原版 Minecraft 中遗漏的小便利功能。

常见的功能包括：

- 可配置的 HUD：护甲状态、坐标、FPS 计数器、药水计时器。
- 提升帧率和减少卡顿的性能选项。
- 额外的快捷键和快速切换，用于经常执行的操作。
- 化妆和视觉设置，例如自定义准星或缩放。

重点在于便利性。 实用客户端不会给你游戏不允许的能力。

## 运行方式

实用客户端通过 [模组加载器](/zh/learn/what-is-a-minecraft-mod-loader) 加载。 在 [Fabric](https://fabricmc.net/) 侧，它是一个正常的 Fabric 模组：你安装适用于你的 Minecraft 版本的加载器，将客户端 jar 文件放入 mods 文件夹，并在启动时加载。 从那时起，它会在你已有的游戏之上添加其功能和配置屏幕。

由于它运行在加载器上，实用客户端遵循与任何其他模组相同的安装路径，并且可以与兼容的模组一起运行。 许多模组也依赖于加载器的共享 API，例如 [Fabric API](/zh/learn/what-is-the-fabric-api)，以干净地钩入游戏。

## 实用客户端与作弊客户端

界限在于客户端是否会给你不公平的优势.

| | 实用客户端 | 作弊客户端 |
| :--- | :--- | :--- |
| 目标 | 方便和生活质量 | 游戏优势 |
| 示例功能 | HUD、FPS 提升、快捷键 | 瞄准辅助、射程、飞行 |
| 服务器规则 | 通常允许 | 几乎总是禁止 |
| 检测风险 | 低 | 高 |

实用客户端旨在被允许使用。作弊客户端旨在违反规则，并被视为作弊行为。有些客户端混合了两种功能，因此最安全的做法是在使用任何附加工具之前检查服务器规则。

## 为什么人们使用它们

吸引力在于获得更流畅的游戏体验，而无需寻找和配置十几个独立的模组。一个安装程序即可提供 HUD、性能选项和快捷键，所有这些都集中在一个设置菜单中。对于竞技玩家来说，干净的 HUD 和稳定的帧率是真正的、允许的优势，这归结于信息和舒适度，而不是违反规则。
