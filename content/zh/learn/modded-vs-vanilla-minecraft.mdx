## 什么是原版 Minecraft

原版 Minecraft 是指未经修改的、直接从 Mojang 获得的原始游戏。当你启动它时，你所体验到的就是标准体验：官方的方块、生物、世界生成和规则，没有任何添加。

原版是所有其他内容的基础。服务器默认期望它，多人游戏与之匹配，并且所有更新首先会在这里发布。如果你从未接触过模组文件夹，那么你正在玩原版。

## 什么是模组版 Minecraft

模组版 Minecraft 是指加载了一个或多个模组的游戏。模组是一组额外的代码，可以改变或添加行为，它通过一个 [模组加载器](/zh/learn/what-is-a-minecraft-mod-loader) 运行，位于 Minecraft 和你的模组之间。

模组覆盖范围很广。有些会添加新的内容，例如方块、物品和维度。有些专注于性能，提高帧率。还有一些是提升游戏体验的工具，可以在不改变游戏规则的情况下改进 HUD 或添加快捷键。你可以选择你想要的模组并一起加载它们。

## 模组是如何实际运作的

你不会编辑 Minecraft 自身的来修改游戏。相反，你安装一个 [模组加载器](/zh/learn/what-is-a-minecraft-mod-loader)，加载器会在启动时在内存中修补游戏。

在 [Fabric](https://fabricmc.net/) 方面，流程很简单：

<Steps>

<Step number={1} title="安装加载器">为您的 Minecraft 游戏版本安装 [Fabric](https://fabricmc.net/) 启动器</Step>

<Step number={2} title="添加你的模组">将模组 jar 文件放入 `mods` 文件夹中</Step>

<Step number={3} title="发射">开始游戏，加载器会将每个模组安装到其中</Step>

</Steps>

[Fabric](https://fabricmc.net/) 是一个轻量级的加载器，因此基础游戏保持接近原版状态，只有您添加的模组才会改变游戏内容。许多模组也依赖于 [Fabric API](/zh/learn/what-is-the-fabric-api)，这是一个共享库，为它们提供干净的游戏钩子。

## 模组版与原版一览

| | 原版 | 模组版 |
| :--- | :--- | :--- |
| 设置 | 无，即装即玩 | 首先安装模组加载器 |
| 内容 | 原始 Mojang 游戏 | 添加方块、物品、工具、调整 |
| 性能 | 固定 | 性能模组可以提高 FPS |
| 多人游戏 | 适用于任何服务器 | 取决于服务器允许的内容 |
| 更新 | 第一天可用 | 等待模组更新 |
| 风险 | 无 | 错误版本或不良模组可能导致崩溃 |

## 您应该玩哪个版本？

如果您想要零设置、游戏发布当天的新版本，或者与任何服务器的保证匹配，请玩原版。它是安全默认选项和共享基础。

如果您想要更好的性能、额外的游戏内容或原版中缺失的质量生活工具，请选择模组版。代价是少量的设置和在 Minecraft 更新后等待模组更新。许多玩家保留单独的配置文件：一个干净的原版安装，以及一个或多个模组版安装.

多人游戏的关键在于服务器。内容模组通常需要匹配的模组服务器，而客户端性能和 HUD 模组通常可以在原版服务器上正常运行。加入游戏前请务必查看规则，确保使用的模组符合要求。
