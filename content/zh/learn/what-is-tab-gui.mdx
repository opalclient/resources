<KeyTakeaways>

- Tab GUI 是一款键盘可导航的菜单，用于浏览和切换 Minecraft 实用工具客户端的模块，无需鼠标。
- 您可以通过按键循环浏览类别（战斗、移动、视觉等），然后选择一个条目以切换它或打开其设置。
- 它可以通过单个按键打开和关闭，并且占据的屏幕空间比浮动式点击 GUI 更小。
- 这是一个客户端菜单，仅在客户端运行。它不与服务器通信，因此本身不构成任何检测风险。
- 一些客户端，例如 Fan，同时提供 Tab GUI 和点击 GUI，并允许您选择使用.

</KeyTakeaways>

Tab GUI 是一种内置于 Minecraft 实用工具客户端的键盘可导航菜单，用于浏览和切换模块，无需触碰鼠标。它不是通过拖动面板在屏幕上移动，而是通过按键循环浏览类别，例如战斗、移动和视觉模组，然后选择一个条目来将其开启、关闭或进入其设置。它是 Click GUI 之外的另一种标准菜单系统，两者都解决了相同的问题：快速访问模块，但输入方式相反。

## 工作原理

Tab GUI 用列表代替了面板。按下菜单的快捷键，它会显示为一个紧凑、可滚动的条目集合，按类别分组：战斗、移动、玩家、视觉，以及客户端组织的其他内容。按键操作在类别之间移动选择，然后在其中一个类别中在模块之间移动，再按一次键可以切换突出显示的模块的开启或关闭状态，或者如果模块有设置，则进入其设置。使用打开它的相同快捷键关闭它，屏幕就会恢复干净。

在任何时候都不会出现光标。这就是整个概念：在这里使用手掌已经放置的按键，完成 Click GUI 使用鼠标所做的一切。

## Tab GUI 与 Click GUI

这两个菜单都指向相同的底层模块列表。改变的是你如何到达那里.

| 方面 | 标签 GUI | 点击 GUI |
|---|---|---|
| 输入 | 键盘 | 鼠标 |
| 屏幕占用 | 紧凑列表，通过一个按键打开和关闭 | 悬浮面板，您可以自由放置和调整大小 |
| 适合 | 快速切换，无需离开 WASD 键 | 视觉浏览，拖动滑块和下拉菜单 |

标签 GUI 奖励那些经常在战斗中途重新配置的玩家，因为通过键盘直接跳转到类别比用光标在嵌套面板中搜索要快。点击 GUI 奖励那些宁愿看到所有内容展开并根据感觉拖动数值的玩家。我们记录到一位客户同时发售了这两种 GUI，让您选择更适合您手型的方式，而不是强迫您使用单一风格.

<Callout variant="tip" title="Opal的版本">
Opal的产品条目中对Tab GUI的描述简洁明了：一个可以通过键盘导航的菜单，用于客户端的功能。它归类于SAFE，与所有改变客户端体验但不影响服务器状态的视觉或生活质量相关的项目属于同一类别。无需鼠标，风险亦无。
