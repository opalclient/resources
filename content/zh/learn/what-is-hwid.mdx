<KeyTakeaways>

- HWID（硬件 ID）是根据您的计算机硬件构建的指纹，用于将一个许可证绑定到一个设备。
- 客户端使用它来阻止许可证共享：一个许可证对应一个设备。
- 重要的硬件更换或操作系统重新安装可能会改变您的 HWID；增加 RAM 或 USB 设备通常不会。
- Opal 允许您在仪表板上每 30 天重置一次 HWID，Discord 提供紧急支持.

</KeyTakeaways>

HWID 代表 **硬件 ID**，它是从您的计算机硬件组件中生成的唯一标识符。您可以将其视为您设备的指纹。当您激活 Opal 等 [客户端](/zh/learn/what-is-a-minecraft-utility-client) 时，您的 HWID 会被记录。从那时起，只有该特定设备才能使用您的许可证。

## 客户端使用 HWID 的原因

HWID 许可解决了简单的问题：**防止许可证共享**。

如果没有 HWID，一个人可以购买一个许可证并与数十位朋友分享。这对需要收入来维护和更新客户端的开发者来说是不利的。HWID 确保一个许可证对应一个设备。

### 生成方式

您的 HWID 通常由以下内容的组合派生：
- CPU 标识符
- 主板序列号
- 磁盘驱动器序列号
- MAC 地址
- 其他硬件标识符

确切的组合因客户端而异。目标是创建一个稳定的标识符，除非您对硬件进行重大修改，否则它不会改变。

## 您的 HWID 何时更改

如果发生以下情况，您的 HWID 可能会更改：
- **更换主要硬件：** 新主板、CPU 或存储驱动器
- **重新安装操作系统：** 一些标识符会重置
- **切换计算机：** 显然是完全不同的 HWID
- **运行某些系统工具：** 磁盘克隆或硬件欺骗

次要更改（添加 RAM、插入 USB 设备）通常不会影响您的 HWID。

## HWID 重置

由于硬件变动（例如升级电脑、更换笔记本电脑或重新安装 Windows），大多数客户端提供 **硬件 ID 重置**.

<Callout variant="info" title="Opal 的 HWID 重置政策">
您可以在账户设置的网页仪表盘中重置您的 HWID。 重置操作每 30 天可用一次。 重置后，在您的新设备上启动 Opal 以绑定新的 HWID。 如果您需要在冷却期前进行紧急重置，请通过 Discord 联系支持。
</Callout>

### 常见重置场景

| 场景 | 需要重置吗？ |
|----------|---------------|
| 新电脑 | 是 |
| 重新安装 Windows | 可能（取决于更改的内容） |
| 新主板/CPU | 是 |
| 添加 RAM | 否 |
| 新 GPU | 通常否 |
| 双启动 | 可能（取决于操作系统） |

## 管理 HWID 的技巧

- **不要浪费重置：** 如果您即将进行硬件升级，请计划在所有更改完成后再进行重置
- **保留仪表板访问权限：** HWID 重置通过 Web 仪表板完成，请确保保存您的帐户凭据
- **边缘情况联系支持：** 如果您已达到 30 天的冷却期但有合法的需求，Discord 支持可以提供帮助

## HWID 与其他授权模式

| 模式 | 工作原理 | 优点 | 缺点 |
|-------|-------------|------|------|
| **HWID** | 与硬件绑定 | 简单，游戏内无需登录 | 硬件更改需要重置 |
| **基于帐户** | 与登录绑定 | 适用于任何设备 | 凭据可能被共享或盗窃 |
| **基于 IP** | 与 IP 地址绑定 | 无需设置 | VPN 和动态 IP 会导致失效 |
| **订阅** | 时间限制 | 为开发者提供收入 | 用户持续成本 |

Opal 使用 HWID 结合帐户身份验证：您需要有效的许可证和匹配的硬件。这在保持用户体验简单的情况下，提供了强大的安全性。
