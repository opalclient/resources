<KeyTakeaways>

- ClickGUI 是 Minecraft 实用工具客户端中，用于鼠标点击切换模块和编辑其设置的点选式菜单。
- 它通过快捷键绑定打开，并显示可拖动、按类别分组的开关、滑块和下拉菜单，其布局会保存到配置文件中。
- 这是一个菜单，而不是游戏玩法功能，因此它本身不构成任何检测风险.

</KeyTakeaways>

点击GUI是Minecraft实用工具客户端中的游戏内菜单，它允许你通过鼠标开关模块并编辑其设置，而无需一次性地在聊天中输入命令。它通过专用的快捷键打开，并以可拖动的、按类别分组的窗口布局呈现客户端的功能集：复选框用于开关，滑块用于数字，下拉菜单用于模式，颜色选择器用于任何视觉元素。对于许多玩家来说，它可能是他们唯一接触的界面，因为它展示了客户端的所有功能，而不是要求你事先知道命令的名称才能使用它。

## 工作原理

点击GUI由一组面板构建而成，通常每个类别一个面板，你可以随意拖动这些面板并在屏幕上折叠或展开。每个面板列出其模块，作为行：一个切换开关用于打开或关闭模块，以及一个箭头用于展开该行以显示其实际设置。展开一个面板，你将获得真实的控件：一个复选框用于简单的开关值，一个滑块用于数字，如范围或速度，一个下拉菜单或循环按钮用于模式，一个颜色选择器用于任何视觉元素。这些控件都不是仅仅为了展示。它们读取和写入模块的实时设置，因此拖动滑块会在你松开鼠标的瞬间改变行为。无需重新打开菜单，无需重新输入命令.

布局在关闭菜单时也不会丢失。大多数客户端会直接将面板位置和打开或关闭状态写入配置文件，因此 GUI 会在会话之间恢复到您离开时的样子。一旦构建出您喜欢的布局，它就会一直保持。

## ClickGUI 与 Tab GUI

一些客户端将 ClickGUI 与第二个纯键盘菜单配对，通常称为 Tab GUI，这是一种有意的划分，而不是冗余的划分。

| | ClickGUI | Tab GUI |
|---|---|---|
| 输入 | 鼠标 | 键盘 |
| 交互 | 拖动面板、点击切换、滑动滑块 | 使用方向键或 Tab 键循环浏览扁平列表 |
| 最适合 | 浏览完整的模块列表、通过感觉调整数值 | 在不离开 WASD 控制的情况下切换模块 |

两者互补，而不是互相替代。ClickGUI 在发现方面更胜一筹：浏览类别、阅读滑块的实际作用、在眼前搜索框的帮助下，通过肉眼调整数值。Tab GUI 在战斗中更胜一筹，当您的手已经在键盘上，伸手去拿鼠标会花费您没有的时间。同时提供这两种 GUI 的客户端押注您会在不同的时刻需要每一种，而且他们通常是正确的.
