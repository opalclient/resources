<KeyTakeaways>

- 晶体光环会自动将末端晶体放置并在目标附近引爆，造成巨大的爆发伤害。
- 它以游戏帧率为基础，运行放置与破坏循环，无需玩家瞄准或点击。
- 它也被称为自动晶体，是晶体 PvP 的自动化形式。
- 它的存在可以通过时机、旋转和距离来检测，这些是人类无法匹敌的.

</KeyTakeaways>

水晶护盾是一种战斗模块，它会自动在目标附近放置终结水晶并将其引爆，造成巨大的爆发伤害。玩家无需瞄准、放置或击中任何东西。该模块会寻找一个有效的方块，放下水晶，然后自行将其破坏。它是一种作弊功能，服务器也会将其视为作弊功能
## 水晶氣場是如何運作的
水晶光环会执行一个与游戏时钟绑定的放置-破坏循环。首先，它会寻找附近的敌人。然后，它会搜索该敌人周围的方块，寻找一个末影水晶可以放置的位置，通常是在黑曜石或基岩之上。它会放置水晶，然后立即攻击它，使其在目标身上爆炸。

末影水晶在原版游戏中具有巨大的爆炸伤害，这使得循环变得危险。如果人为地手动执行此操作，需要放置一个方块、切换到水晶、放置水晶，然后点击它，所有这些都在一秒钟内完成。该模块会自动执行每个步骤，速度远快于人类，因此目标会毫无预兆地受到爆发伤害。

这种战斗方式通常被称为水晶 PvP。水晶光环是它的自动化版本
## 常用设置
水晶光环会暴露一些旋钮，它们会牺牲速度来换取行为的显眼程度.

| 设置 | 控制内容 |
|---|---|
| 放置范围 | 晶体可放置在玩家附近的最大距离 |
| 破坏范围 | 模块引爆已放置晶体的最大距离 |
| 目标范围 | 敌人必须靠近多少才能触发循环 |
| 放置延迟 | 放置晶体之间的等待时间 |
| 破坏延迟 | 引爆前的等待时间 |
| 旋转 | 视角是否转向放置点或破坏点 |
| 伤害检查 | 放置前目标所需的最小伤害，以避免自伤 |

伤害检查很重要，因为末端晶体会伤害靠近爆炸的人，包括施法者。安全的配置仅在目标受到的伤害大于玩家受到的伤害时才放置晶体。

## 为什么会被检测到

水晶光环会被检测到，因为它比人类更快、更精确地放置和破坏。服务器端的反作弊程序会监控放置和破坏的时间过于紧凑、玩家从未查看过的晶体互动，以及放置或破坏的距离超过正常限制。这些都是自动化循环而非真实鼠标和键盘的标志.

放置水晶是容易的部分。真正的工程难题在于伪装这个循环，因为放置-破坏的节奏和视野的摆动方式必须看起来像是人类输入，否则现代反作弊系统就会标记出这个模式。这就是为什么许多版本会增加延迟、随机性和一个在服务器允许的距离内的范围上限。

## 水晶光环与其它战斗模块

水晶光环与类似[杀戮光环](/zh/learn/what-is-killaura)的近战自动攻击者在自动化内容上有所不同。杀戮光环直接瞄准并攻击实体。水晶光环通过放置和引爆末影水晶来战斗，造成爆炸伤害。两者都运行相同的移动、瞄准和计时循环，而无需玩家输入，这就是为什么反作弊系统会对两者都施加类似的计时和旋转检查。该模块在不同的客户端中也被追踪为“自动水晶”.
