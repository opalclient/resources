In PvP, a steady frame rate matters more than a high one. A game that holds a consistent number feels far more responsive than one that spikes and dips mid-fight. The recipe is [Sodium](https://modrinth.com/mod/sodium) for raw performance, a sensible frame cap for consistency, and a few settings tuned for clarity over eye candy. One note up front: Lithium is not compatible with Opal, so if you run Opal, lean on Sodium for the headroom instead.

<KeyTakeaways>

- Steady frametime beats peak FPS in a fight. Aim for a number you always hold.
- Sodium is the base; it gives the headroom stable frames need.
- Lithium smooths game logic but is not compatible with Opal. Skip it if you run Opal.
- A frame cap plus clarity settings finishes the job.

</KeyTakeaways>

## Build the performance core

[Sodium](https://modrinth.com/mod/sodium) rewrites Minecraft's renderer and is the single biggest gain for stable frames. Put the [Fabric](https://fabricmc.net) loader and [Fabric API](https://modrinth.com/mod/fabric-api) in place, then drop Sodium in your `mods` folder. It runs fine alongside Opal.

<Callout variant="warning" title="Running Opal? Skip Lithium">
Lithium is not compatible with Opal. Use Sodium for your framerate and leave Lithium out of the folder. Sodium covers the rendering side, which is the larger win for PvP anyway.
</Callout>

## Cap your frame rate

A frame that arrives on a steady schedule feels better than a faster but erratic one. Set a cap your machine can hold comfortably at all times rather than chasing the highest possible number. Consistent frametime is the goal, not a big number in the corner.

## Tune for clarity

<Checklist>
<ChecklistItem>Render distance: moderate, so distant chunks do not cause dips mid-fight.</ChecklistItem>
<ChecklistItem>Graphics: Fast.</ChecklistItem>
<ChecklistItem>Particles: minimal, so effects do not clutter the screen or cost frames.</ChecklistItem>
<ChecklistItem>Smooth lighting: low.</ChecklistItem>
</Checklist>

<Callout variant="tip" title="Quick win">
Dropping render distance from 16 to 12 chunks hands back a large chunk of your framerate with almost no visual cost in close-range PvP.
</Callout>

## Where a PvP client fits

Stable frames are the foundation; the client is what you do with them. Opal is a Fabric mod for Minecraft 1.21.10, so it installs into the same `mods` folder as Sodium and adds 200+ features across combat, movement, visual, and utility, behind a glass UI. It is a full client, so like any large mod it uses some resources while it runs. That is exactly why Sodium matters: it gives you the headroom to run a feature set like Opal's and still hold steady frames in a fight.
