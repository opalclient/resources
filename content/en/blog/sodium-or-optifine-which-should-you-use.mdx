For raw frame rate on modern Minecraft, [Sodium](https://modrinth.com/mod/sodium) (with [Iris](https://modrinth.com/mod/iris) for shaders) is the pick. For a single all-in-one jar with built-in zoom, connected textures, and shaders, OptiFine is the convenient classic. Most performance-focused players now run Sodium. And if you plan to run a [Fabric](https://fabricmc.net) client like Opal, the decision is already made: Opal lives in the Fabric and Sodium ecosystem.

<KeyTakeaways>

- Sodium delivers the highest FPS and updates to new versions fast.
- OptiFine is the one-jar all-in-one, but its raw FPS often trails Sodium.
- Sodium is modular: add Iris for shaders, plus a Fabric client like Opal.
- Want to run Opal? Use Sodium. It is the Fabric rendering path Opal is built for.

</KeyTakeaways>

## Sodium

A modern rendering rewrite focused purely on performance, built for [Fabric](https://fabricmc.net). It generally delivers the highest FPS and gets new Minecraft versions quickly. It does one job well, and you add other mods alongside it: [Iris](https://modrinth.com/mod/iris) for shaders, and a client like Opal for features.

## OptiFine

The long-standing all-in-one. One jar gives a performance boost plus zoom, connected textures, dynamic lighting, and shader support. The trade-offs: its raw FPS often trails Sodium, it can lag behind on the newest versions, and it does not always play nicely with a Fabric mod setup.

## Where Opal fits

Opal is a Fabric mod for Minecraft 1.21.10, so it runs in the same ecosystem as Sodium and Iris: drop all three in the `mods` folder and you get high FPS, optional shaders, and a full client with 200+ features and a glass UI. That is the Sodium path. If you were leaning OptiFine purely for zoom or convenience, note that a Fabric setup with Sodium gives you the framerate and room for the client too.
