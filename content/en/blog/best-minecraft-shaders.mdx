The safest all-round pick is **Complementary**, which looks great and stays reasonably light. Run any shader pack through [Iris](https://modrinth.com/mod/iris) and [Sodium](https://modrinth.com/mod/sodium) for the best performance. Below are the packs worth knowing, from balanced to heavy, so you can match one to your hardware and taste, and they all run fine alongside Opal.

<KeyTakeaways>

- Complementary is the safe default: good looks, sensible performance.
- BSL, SEUS, and Sildur's Vibrant scale from light to heavy.
- Run every pack through Iris on Sodium for the best framerate.
- Shaders run with Opal; only Lithium is incompatible.

</KeyTakeaways>

## Complementary

The popular default. Balanced lighting, good colors, and sensible performance, with profiles that let you trade looks for frames. A safe first shader for most players.

## BSL

Clean, bright, and highly tunable, with a big settings menu. A long-standing favorite that scales from light to fairly heavy depending on how you configure it.

## SEUS

Known for a polished, cinematic look. The heavier variants lean realistic and demand a stronger GPU; lighter versions exist for weaker machines.

## Sildur's Vibrant

Comes in tiers from very light to high. The lite profiles add color and shadows without a big frame cost, which makes this a go-to for mid-range setups.

## How to run them

<Steps>

<Step number={1} title="Install Iris and Sodium">
[Iris](https://modrinth.com/mod/iris) runs the shaders, [Sodium](https://modrinth.com/mod/sodium) keeps the framerate. Put the [Fabric](https://fabricmc.net) loader and [Fabric API](https://modrinth.com/mod/fabric-api) in first.
</Step>

<Step number={2} title="Add the pack">
Drop the shader pack in the `shaderpacks` folder and select it in video settings.
</Step>

</Steps>

<Callout variant="tip" title="Opal users">
Sodium and Iris both run with Opal, so you can keep shaders on with the client installed. Leave Lithium out, since it is not compatible with Opal.
</Callout>

## Shaders plus a client

Opal is a Fabric mod for Minecraft 1.21.10, so it sits in the same `mods` folder as Iris and Sodium. Its glass UI is built to read clearly over a shader-lit world, so a good shader pack and the client complement each other: the world looks the part, and the interface stays crisp on top of it.
