## A decade apart in Minecraft versions

Fan runs on MC 1.8.9. Opal v2 runs on MC 26.2. Those two Minecraft releases are roughly a decade apart, and a client built for one does not run on servers expecting the other. This is not really a UI-versus-UI, feature-versus-feature comparison so much as two clients from different eras of the same scene. If you are still on 1.8.9 for whatever reason, Opal is not an option for you at all.

## One of these is still getting updates

Fan's own record lists it as abandoned, with an active period running roughly 2020 through 2021 on the intent.store marketplace. Nothing in that record points to ongoing development past that window. Opal v2 is the opposite case: it is the current, actively maintained Opal client, tracking MC 26.2 with an update logged this year. A client that stops getting updates stops keeping up with whatever the server side changes next. That is true for any client, not just this pairing.

## Module count and design philosophy

Fan shipped 55 modules across seven categories under a single-author `cedo/` namespace, credited primarily to Cedo. Some of its design choices are genuinely clever: a dedicated "hidden" category that keeps certain modules off the in-client Array List HUD, and a twin Scaffold / BadScaffold pair shipped side by side for bypass A/B testing. It also ran a Click GUI and a TabGUI at once, letting the user pick their own navigation style. That is real design thinking from a single developer, not an accident.

Opal v2 ships 70 modules split into a blatant set (flight, scaffold, aggressive combat tuned for BedWars/SkyWars-style minigame combat) and a quieter set built to hold up on most servers, plus an embedded GraalJS scripting runtime for anyone who wants to extend it. It is a larger, more current feature set built around a single custom glass UI engine rather than two competing menus, if you would rather not choose between them yourself.

## What you are actually getting for the money

Opal v2 costs $8.99 and comes through a launcher: log in and play, no manual setup. Fan's record does not list a price, only that it was sold on intent.store, and the distributed source has no build configuration at all, just an extracted snapshot with a hand-written `Start.java` entry point. Interesting for anyone studying the client's history. Less fun if you just want to load into a game tonight.

Credit where it is due: Fan's hidden-module category and its twin-Scaffold testing setup are good ideas that hold up as design patterns, and the dual-GUI approach is a real answer to "let players pick their own navigation." A single-developer client that shipped that much thought into UX details is worth studying even now. None of that changes the fact that it has not been updated since 2021 and does not run on the Minecraft version most people are actually playing. If you want something maintained, on the current release, with active updates and a modern UI, that is Opal.
