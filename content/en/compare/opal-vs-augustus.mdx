Augustus only supports **Minecraft 1.8.9**. That's it. No 1.12, no 1.19, no 1.21. In 2026, the majority of servers and the broader Minecraft ecosystem have moved to modern versions.

Opal targets **Minecraft 1.21.10** via Fabric and is compatible with both 1.8-style and latest-version servers. You're not locked into a version from 2014.

## Features

Augustus does have an impressive module count — 240+ modules with 38,000+ settings. It's extremely customizable for the servers it supports.

Opal has 200+ features across 69 modules. While the raw number is slightly lower, Opal covers all the essential categories (combat, movement, visual, utility, world) and adds **ghost features** that Augustus doesn't have. Plus, Opal's GraalVM scripting engine lets you create custom automation that extends the client beyond its built-in modules.

## Anticheat Support

One area where Augustus has an edge: it claims bypasses for **Intave, Polar, Grim, and Vulcan** anticheats that Opal doesn't target.

Opal focuses on **popular server bypasses** and ghost features that work everywhere. If you need multi-anticheat blatant support, check out **Terminus** — available as an upgrade for Opal users at $36.99.

## Platform

Augustus is a standalone client with no ecosystem around it. No launcher, no dashboard, no config browser, no forums.

Opal comes with the full package — and it's all included in the $8.99 price.

## The Verdict

Augustus made sense when 1.8.9 was the standard and customization was king. In 2026, paying $85 for a single-version client with no platform is hard to justify.

Opal delivers modern version support, hybrid features, a full ecosystem, and a scripting engine — all at **$8.99**. If you need the specific additional anticheats Augustus supports, consider the Opal + Terminus path for a combined cost still well under Augustus's price.
