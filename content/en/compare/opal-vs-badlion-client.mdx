## What you're actually installing

Badlion Client is a standalone application. It is not a mod you drop into an existing Fabric or Forge setup, so there is no loader version to chase down or match against your Minecraft install. That is a genuine point in its favor if you would rather never think about mod loaders at all: Badlion owns its whole pipeline end to end, and being standalone means it is not tied to Fabric's release cadence or the compatibility breakage that comes with it.

Opal is a Fabric mod under the hood, but it ships through its own launcher. You are not manually installing Fabric Loader or hunting down a matching Fabric API build; the launcher handles that plumbing for you. In practice, setup friction on either side is close to zero. Architecturally they are solving the same problem two different ways, and if "no mod loader in the picture, ever" is a hard requirement for you, Badlion's model gets you there more directly.

## Feature depth: documented vs undocumented

Here is the part where we have to be honest instead of persuasive. Our note on Badlion Client is a stub: active status, written in Java, installs standalone. That is the entire record. Opal, being our own product, we can spec out in full. 70 modules split across combat, movement, visual, world, and utility. A KillAura with separate target-selection and aim-assist layers. An ESP family covering blocks, entities, and items. Scaffold for bridging. Movement tools including Flight, LongJump, Speed, and NoSlow. A Dynamic Island style HUD with keystrokes, session stats, and a Spotify now-playing panel. The interface runs on a custom glass UI engine built on Blaze3D (blur, fresnel, refraction, and shadow effects), with a GraalJS and Yoga runtime underneath handling layout and animation, and a full embedded GraalJS scripting engine sits behind all of it, so you can script directly against the client instead of waiting on the next module release. Alongside the obvious BedWars/SkyWars-style combat tools there is a quieter layer of tools built to hold up across most servers rather than announce itself.

Whether Badlion matches, beats, or falls short on any of that, we genuinely cannot tell you. Neither can anyone else without a source, and we would rather leave the gap blank than fill it with a guess.
