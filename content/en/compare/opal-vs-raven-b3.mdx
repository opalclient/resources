## Which Minecraft version you're actually playing

That's the real question here, not which client is "better." Opal runs on MC 26.2 through Fabric. Raven B3 runs on MC 1.8.9 through Forge. There's no overlap between the two, so there's no real choice to make until you know which game you're actually launching.

If you're still on 1.8.9, for old PvP servers, legacy anarchy, whatever keeps people on that version, Opal simply doesn't target that client version at all. Raven B3 is what runs there. If you're on current Minecraft, Raven B3 can't follow you. It stays exactly where its build was frozen back in the 1.8.9 era.

## What's actually documented about each client

Opal's record is the kind you can build a spec sheet from: 70 modules split across combat, movement, visual, world, and utility categories, a custom glass UI engine, and an embedded GraalJS scripting engine you can load and unload scripts into while the client is running. Flight and scaffold are two of the specific modules on offer, covering movement and bridging respectively, and the combat set ships its own aim-assist targeting alongside a KillAura module.

Raven B3's record is thinner: a Forge mod for MC 1.8.9, part of the Raven lineage, built by blowsy, marked legacy. No module list, no price, no UI details survive in what's citable about it. That's not a knock on the client itself, the 1.8.9 scene has produced plenty of clients worth respecting. It's just a statement about how little of this one is written down anywhere we can point to. We're not going to guess at features it might have, and blowsy's work deserves better than a made-up spec sheet.

## Installing a mod yourself vs a launcher

Opal ships through a launcher: log in, it pulls the current build, you play. That's the entire setup, and it's how the client stays current on MC 26.2 without you doing anything else.

Raven B3, per its own record, is a Forge mod: install Forge, drop the jar in, manage it the way client jars from that era generally worked. Its legacy status means there's no update cycle running behind it anymore, so whatever build you find is the one you get. If you already know your way around a 1.8.9 mods folder, that's a five-minute job. If you don't, "install as a Forge mod yourself" is doing a lot of quiet work in that sentence.

## Which Raven you're even looking for

Worth flagging before you go searching further: Raven B3 isn't the only Raven-named client out there.

<Callout variant="warning" title="Two different Ravens">
The client documented here is Raven B3, by blowsy. Raven B++ is a separate, unrelated client by K-ov. Same naming pattern, different projects, different lineages. If a search result mixes the two up, they aren't the same thing.
</Callout>
