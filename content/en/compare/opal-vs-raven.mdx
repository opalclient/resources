## Active development vs a legacy name

Opal is in active development on MC 26.2. It ships as a Fabric mod, gets updates, and comes backed by a launcher and a community support channel.

Raven is documented as legacy. There is no current version, no storefront, and no update cadence attached to the name. What keeps Raven recognizable is reputation built up over years in the PvP and ghost-client scene, not an active release schedule, and reputation alone does not install a mod, patch a bug, or answer a support ticket.

## A Fabric mod with a launcher vs an external injector

Opal loads as a Fabric mod through its own launcher. Log in, it installs itself, and you are in-game. That is a modern injection model built for a modern loader.

Raven's own documentation lists its injection model as external, meaning it hooks into an already-running Minecraft process rather than loading through a mod loader. That is an older, more hands-on delivery style, and nothing in the documentation walks you through it: no launcher, no one-click install, no setup guide. If you already know how to run an external injector, fine. If you do not, you are teaching yourself before you have even opened the game.

## What $8.99 buys that free archives don't

Raven does not have a listed price, mainly because it is not really for sale anymore. What is left of it is scripts and source that circulate through community archives, which is a genuinely useful way to preserve a piece of client history. It is just not a way to get a supported product.

Opal's $8.99 buys an actively maintained client instead: a launcher that keeps it current, a UI built for MC 26.2, and a module set, 70 of them, split across combat, movement, visual, world, and utility, that was tuned on purpose rather than assembled from whatever scripts happened to survive.

## The ghost-client reputation, and where the documentation stops

Raven earned its name in the ghost-client scene, and that reputation is real. Still being talked about years after going legacy is not nothing, and it is a genuine point in Raven's favor that an active paid product like Opal does not get to claim for itself: you cannot buy that kind of word of mouth. What the documentation does not give us is a feature list, a module count, or a Minecraft version to set next to Opal's, so this page stops at what is actually documented instead of guessing at the rest.
