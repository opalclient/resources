Auto Build is a world module that constructs a predefined structure or schematic automatically, working through a saved block-by-block layout without the player placing anything by hand. Load a structure, and the module places every block itself, pulling from whatever matching blocks you're carrying. It's a building tool first. Nothing about how you fight or move changes, only how fast a wall goes up.

## How it works

Auto Build starts from a predefined structure or schematic: a saved layout that specifies exactly which block goes where. The module works through that layout position by position, placing the correct block at each spot from whatever matching blocks sit in your inventory. You never aim, click, or walk to the location yourself. The placement sequence runs on its own.

Because the structure is fixed ahead of time, the module has no decisions left to make. It's executing a known list of coordinates and block types, and it can do that faster and more consistently than a person clicking every face by hand.

Wurst, the client that documents this module, files it under building and world interaction rather than combat. That's the whole appeal: a construction shortcut for towers, pyramids, and other multi-block layouts that take forever to place one at a time by hand, where a mistake partway through normally means fixing it yourself. Auto Build never misplaces a block and never loses track of where it is in the layout.

## Is it safe to use

Auto Build isn't a stealth PvP module the way KillAura or Scaffold are. It doesn't touch reach, aim, or hit timing, and most people run it for building rather than fighting, so it doesn't carry the same detection profile as a combat cheat.

It's still automated block placement, though, and that alone can be enough. A server watching placement rate or pattern for anti-grief or anti-cheat reasons can treat a structure going up faster than a human could build it as suspicious, particularly on survival multiplayer. In single-player or creative-focused settings, where building tools are already expected, that risk mostly disappears.

Opal already ships other world-category tools of its own, catching a bad fall before it becomes a death is one of them, but a schematic-executing build bot like this isn't part of the lineup.
