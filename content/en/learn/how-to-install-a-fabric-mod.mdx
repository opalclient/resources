<KeyTakeaways>

- Installing a Fabric mod is three steps: loader, Fabric API, then the mod jar.
- Every piece (loader, API, mod) must target the same Minecraft version.
- Mods live in the mods folder inside your Minecraft directory.
- Most install failures are a version mismatch or a missing Fabric API.

</KeyTakeaways>

Installing a [Fabric](https://fabricmc.net/) mod takes three steps: install the Fabric loader for your exact Minecraft version, put the Fabric API in your mods folder, then drop the mod's jar in beside it. Launch the Fabric profile and the mod loads. Most install problems come from a version mismatch, so match every piece to the same Minecraft version.

## The three steps

<Steps>

<Step number={1} title="Install the Fabric loader">
Install the [Fabric](https://fabricmc.net/) loader for your Minecraft version, using the official Fabric installer or a launcher that supports Fabric. See [how to install the Fabric loader](/en/learn/how-to-install-fabric-loader) for the full walkthrough.
</Step>

<Step number={2} title="Add the Fabric API">
Place the [Fabric API](/en/learn/what-is-the-fabric-api) jar in your mods folder. Most mods depend on it.
</Step>

<Step number={3} title="Add the mod">
Place the mod's jar in the same mods folder, next to the Fabric API.
</Step>

</Steps>

Then launch the Fabric profile and the mod loads on startup. If it does not appear, check the troubleshooting list below.

## Where the mods folder is

| OS | Path |
| :--- | :--- |
| Windows | `%appdata%\.minecraft\mods` |
| macOS | `~/Library/Application Support/minecraft/mods` |
| Linux | `~/.minecraft/mods` |

If the `mods` folder does not exist yet, run the Fabric profile once and it gets created for you.

## When a mod will not load

<Checklist>
<ChecklistItem>Mod does not appear: the jar is in the wrong folder, or you launched a non-Fabric profile.</ChecklistItem>
<ChecklistItem>Crash on launch with a missing-class error: the Fabric API is missing or the wrong version.</ChecklistItem>
<ChecklistItem>Version mismatch: the loader, the Fabric API, and the mod must all target the same Minecraft version.</ChecklistItem>
</Checklist>
