<KeyTakeaways>

- Opal ships 70 modules across six categories: Combat, World, Movement, Utility, Visual, and Other.
- 21 are safe, cosmetic changes that only touch your own client. 49 are use-at-own-risk features that change gameplay in ways anticheats can catch.
- Every module below links to a full breakdown of how it works and why it does or doesn't get detected.
- This page is the map. Start here, then go deep on whatever you're actually trying to do.

</KeyTakeaways>

A module is a single feature you can flip on inside Opal, from something as quiet as a Discord status update to something as loud as an aimbot. Some are pure convenience. Some will get you banned on a server that's paying attention. Knowing which is which matters more than knowing they exist, so this page sorts all 70 by category and tags every one of them safe or use-at-own-risk before you touch a single toggle.

## Safe vs. use-at-own-risk, in one table

| | Safe | Use at own risk |
|---|---|---|
| **What it touches** | Only your own client's rendering or convenience | Movement, combat, or timing the server can observe |
| **Anticheat exposure** | None, nothing unusual is sent to the server | Real, these are exactly what anticheats are built to catch |
| **Examples** | Full Bright, Click GUI, Discord RPC, Streamer Mode | Kill Aura, Scaffold, Speed, Reach |
| **Where it's fine to run** | Any server, any time | Depends entirely on how seriously that server enforces its rules |

The label isn't a promise, it's a starting point. Read the dedicated page for any module you're actually planning to use before you flip it on.

## How to actually pick modules

Don't turn everything on. Start with whatever solves the specific problem you have: Full Bright if caves are wasting your time, Auto Tool and Auto Armor if inventory management is the boring part of your day, Kill Aura only if you've already accepted the ban risk that comes with it. Stack safe modules freely. Treat every use-at-own-risk module as a decision, not a default.

<Callout variant="tip" title="Where to start">
New to Opal? The safe column above is a genuinely good place to spend your first session: Click GUI to see what's available, Full Bright and Streamer Mode if you stream, Inventory Manager if clutter bothers you. None of it risks the account.
</Callout>
