Opal include un motore JavaScript GraalVM, un runtime JS completo in esecuzione all'interno del client di Minecraft. Scrivi script che interagiscono con le API interne di Opal mentre il gioco è attivo. Non si tratta di un registratore di macro o di un sistema di bind dei tasti. È un vero ambiente di programmazione con accesso al sistema eventi del client, alle configurazioni dei moduli e allo stato del gioco.

<KeyTakeaways>

- Un runtime JavaScript GraalVM completo, non un semplice registratore di macro.
- Rispondi agli eventi di gioco, modifica le impostazioni dei moduli e disegna il tuo HUD.
- Gli script vengono caricati a runtime, senza riavvii, ed eseguiti in un ambiente sandbox rispetto alle API di Opal.
- La maggior parte degli altri client si fermano a macro di base.

</KeyTakeaways>

## Perché GraalVM
GraalVM è un runtime poliglotto di Oracle. L'abbiamo scelto per ragioni concrete:

- Velocità: la compilazione JIT mantiene gli script sufficientemente veloci da poter essere eseguiti ad ogni tick.
- JavaScript: la maggior parte delle persone lo conosce già, quindi la curva di apprendimento è breve.
- Thread safety: i binding da JS a Java gestiscono la concorrenza correttamente, invece di lasciarti inciampare.
- Modelli familiari: le API JavaScript standard si comportano come ci si aspetta
## Cosa puoi costruire

### Automazione personalizzata
Rispondi agli eventi di gioco:

```javascript
// Auto-respond to certain chat messages
on('chat', (event) => {
  if (event.message.includes('1v1?')) {
    sendChat('/duel accept');
  }
});
```
### Configurazione dinamica dei moduli
Modifica le impostazioni del modulo in base allo stato di gioco:

```javascript
// Increase reach when health is low
on('tick', () => {
  const health = player.getHealth();
  if (health < 8) {
    modules.get('Reach').setSetting('distance', 3.3);
  } else {
    modules.get('Reach').setSetting('distance', 3.1);
  }
});
```
### Elementi HUD personalizzati
Crea i tuoi overlay:

```javascript
// Display a combo counter on screen
let combo = 0;
on('attackEntity', () => { combo++; });
on('damage', () => { combo = 0; });
on('render2d', (ctx) => {
  ctx.drawText(`Combo: ${combo}`, 10, 10, 0xFFFFFF);
});
```
### Profili specifici per server
Passa automaticamente alle configurazioni di gioco quando entri in server diversi:

```javascript
on('serverJoin', (event) => {
  if (event.ip.includes('pvp.example.com')) {
    loadConfig('server-bypass');
  } else {
    loadConfig('ghost-default');
  }
});
```
