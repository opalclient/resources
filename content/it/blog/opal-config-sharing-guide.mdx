Una configurazione in Opal è un'istantanea salvata di tutte le impostazioni dei tuoi moduli: quali moduli sono attivi, i loro valori, i tasti di scelta rapida e i parametri. Considerala come un equipaggiamento che salvi, condividi e alterni. Il browser di configurazione di Opal la trasforma in una libreria comunitaria, così puoi essere competitivo su un nuovo server in pochi secondi invece di doverlo configurare da zero.

<KeyTakeaways>

- Una config è un loadout completo: moduli, valori, keybind, parametri.
- Esplora, cerca e ordina le config della community dalla dashboard.
- Salva le tue, poi pubblicale sul browser pubblico con un solo click.
- Usa gli script per caricare automaticamente la config giusta per ogni server.

</KeyTakeaways>

## Esplorazione delle configurazioni pubbliche

Il browser di configurazioni nella dashboard di Opal è dove puoi trovare le configurazioni condivise dalla community. Apri Configurazioni, esplora l'elenco pubblico, cerca per nome e ordina per Più recenti, Più scaricate o Più apprezzate.

<Callout variant="tip" title="Come dovrebbe essere una buona configurazione">
Un nome descrittivo come "Competitive Bedwars Ghost" o "Blatant Speed Bridge", un buon numero di download e "mi piace", e una data di caricamento recente. Più recente è, più è probabile che funzioni con i bypass attuali.
</Callout>

## Scaricare le configurazioni

Ti è piaciuta una? Aprila, leggi i dettagli e scaricala. Si salva nella tua collezione privata e si carica in gioco.

Per caricarla, usa il comando di configurazione:

```
/load config-name
```

Oppure apri il modulo Config Browser in gioco e scegli tra le tue configurazioni salvate.

## Creare le tue

<Steps>

<Step number={1} title="Regola le impostazioni">

<Step number={3} title="Pubblicalo (opzionale)">Sulla dashboard, apri Configs, trova la configurazione salvata, clicca su Pubblica e dagli un nome chiaro. Le configurazioni pubblicate compaiono nel browser pubblico</Step>

</Steps>

## Strategie di configurazione

### Specifiche per server

- Server Ghost: assistenza mirata sottile, portata 3.1, velocità 85%.
- Server Blatant: suite di bypass completa per giochi casuali.
- Practice PvP: funzionalità ghost solo, ottimizzate per i duelli.

### In base allo stile di gioco

- Aggressivo: portata più alta, riduzione della velocità più forte, attacchi più veloci.
- Difensivo: velocità e movimento prioritari, assistenza mirata minima.
- Equilibrato: moderato in tutti gli aspetti.

### Passaggio rapido

Utilizza il [motore di scripting](/it/blog/introducing-opal-scripting-engine) di Opal per caricare automaticamente una configurazione in base al server a cui ti unisci, in modo che la configurazione corretta sia sempre attiva senza doverla modificare manualmente.

## Perché la condivisione delle configurazioni è importante

Una piattaforma di configurazioni integrata è rara. Con la maggior parte dei client, si scambiano file JSON su Discord o si indovinano le impostazioni da un video. Il sistema di Opal permette ai nuovi utenti di essere competitivi fin da subito con una configurazione collaudata, agli utenti esperti di condividere ciò che funziona e a tutti di beneficiare di impostazioni testate dalla comunità. È parte di ciò che rende Opal una piattaforma piuttosto che un semplice client.
