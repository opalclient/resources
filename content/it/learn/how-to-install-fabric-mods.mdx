<KeyTakeaways>

- Fabric è un mod loader leggero; installa prima Fabric Loader, quindi trascina i file JAR dei mod nella cartella mods.
- Per Minecraft 1.21, ti serve Java 21 o superiore, e Minecraft deve essere stato avviato almeno una volta prima dell'installazione.
- I file JAR dei mod vanno in %appdata%\.minecraft\mods su Windows (con equivalenti per macOS e Linux).
- Opal utilizza un sistema di loader: l'Opal Launcher gestisce per te Fabric, l'autenticazione, l'installazione e gli aggiornamenti.

</KeyTakeaways>

Fabric è un mod loader leggero per Minecraft. È veloce, attivamente mantenuto e supporta le versioni moderne di Minecraft. Molti client di utilità, tra cui Opal, funzionano su Fabric. Questa guida illustra l'installazione di Fabric Loader, l'aggiunta di mod e la configurazione di Opal
## Prerequisiti
Prima di iniziare, assicurati di avere:

- **Minecraft Java Edition** installata e lanciata almeno una volta
- **Java 21** o versione più recente installata (richiesta per MC 1.21+)
- Una connessione internet funzionante
## Installa Fabric Loader

<Steps>

<Step title="Scarica il programma di installazione di Fabric">

Vai sul sito ufficiale di Fabric all'indirizzo **fabricmc.net**, clicca su **Download**, seleziona l'opzione **Installer** e scarica il file JAR di installazione universale

</Step>

<Step title="Esegui il programma di installazione e conferma le impostazioni">

Esegui il file JAR di Fabric installer che hai scaricato, quindi assicurati che queste impostazioni siano corrette:

- La scheda **Client** è selezionata
- **Versione di Minecraft:** 1.21.10 (o la versione desiderata)
- **Versione del Loader:** Ultima versione stabile (0.17.x consigliata)
- **Percorso di installazione:** La tua cartella `.minecraft` predefinita

Clicca su **Installa**. Dovresti visualizzare un messaggio di successo

</Step>

<Step title="Verifica l'installazione">

Apri il Minecraft Launcher, clicca sulla scheda **Installazioni** e cerca un nuovo profilo chiamato **fabric-loader-1.21.10** (o simile). Selezionalo e clicca su **Play**. Se Minecraft si avvia con il logo di Fabric nella schermata del titolo, allora va tutto bene

</Step>

</Steps>

## Installa le Mod

Le mod di Fabric vanno inserite nella cartella `mods`:

- **Windows:** `%appdata%\.minecraft\mods`
- **macOS:** `~/Library/Application Support/minecraft/mods`
- **Linux:** `~/.minecraft/mods`

Se la cartella `mods` non esiste, creala. Per installare una mod:

<Steps>

<Step title="Scarica il file JAR della mod">

Scarica il file `.jar` della mod dalla sua fonte

</Step>

<Step title="Mettilo nella cartella dei mods">

Sposta il file `.jar` nella cartella `mods` per la tua piattaforma

</Step>

<Step title="Avvia con il profilo Fabric">

Avvia Minecraft utilizzando il profilo Fabric, non quello vanilla

</Step>

</Steps>

## Installazione di Opal Client

Opal utilizza un **sistema di caricamento**: non devi inserire l'intero client nella cartella dei mod. Invece:

<Steps>

<Step title="Acquista Opale">

Acquista Opal dalla plancia ($8.99 a vita)

</Step>

<Step title="Scarica Opal Launcher">

Scarica **Opal Launcher** dalla pagina dei download. Si occupa automaticamente di autenticazione, installazione e aggiornamenti

</Step>

<Step title="Oppure usa il JAR del loader">

In alternativa, scarica il file JAR di Opal Loader e posizionalo nella cartella `mods`. Si autenticherà e scaricherà il client a runtime

</Step>

</Steps>

Il launcher è l'approccio consigliato: gestisce tutto per te.

## Risoluzione dei problemi

### "Fabric non compare nel launcher"
Assicurati di aver installato Fabric per la versione corretta di Minecraft. Riavvia l'installer se necessario.

### "Il gioco si blocca all'avvio"
- Verifica di avere installato **Java 21** (non Java 8 o 17)
- Rimuovi eventuali mod incompatibili dalla cartella `mods`
- Assicurati che Fabric Loader sia aggiornato

### "Le mod non si caricano"
- Verifica che i file `.jar` siano nella cartella `mods` corretta
- Assicurati di avviare il gioco con il **profilo Fabric**, non vanilla
- Controlla che la versione della mod corrisponda alla tua versione di Minecraft

### "Opal non si autentica"
- Assicurati di avere una connessione internet attiva
- Verifica che la tua chiave di licenza sia attivata nella dashboard
- Controlla che il tuo HWID non sia cambiato (il reset è disponibile ogni 30 giorni nella dashboard)
- Gli utenti in regioni con restrizioni internet potrebbero aver bisogno di un proxy: controlla la pagina di acquisto per i dettagli
