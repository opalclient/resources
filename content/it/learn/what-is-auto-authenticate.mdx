<KeyTakeaways>
- AutoAuthenticate è un modulo che esegue automaticamente il login sui server di auth-plugin rispedendo la tua password salvata come comando di login o registrazione.
- Sostituisce solo la digitazione, non il gameplay, quindi non ha effetti sul combattimento, sul movimento o su qualsiasi cosa tracciata da un anticheat, e appare identico alla digitazione di un giocatore.
- Presente in client come Tenacity, classificato come miglioramento della qualità della vita piuttosto che come strumento di combattimento.
</KeyTakeaways>

**AutoAuthenticate** è un modulo client di utilità per Minecraft che ti fa accedere automaticamente a un server non appena ti connetti, inviando la tua password salvata come comando di login o registrazione in chat. Se hai mai giocato su un server craccato o in offline-mode, sai già come funziona: ti connetti, digiti `/login la_tua_password`, e poi inizi effettivamente a giocare. AutoAuthenticate evita il secondo passaggio facendolo per te, istantaneamente, ogni volta.

## Come funziona

Molti server che utilizzano account craccati o in offline-mode si proteggono con un plugin di autenticazione. Prima di poterti muovere, chattare o interagire con qualsiasi cosa, devi eseguire un comando di login (se hai già un account) o un comando di registrazione (al primo accesso), entrambi con una password associata. Se ti dimentichi questo passaggio, rimani bloccato fino a quando non lo digiti correttamente.

AutoAuthenticate memorizza una password per ogni server e monitora il momento esatto in cui la tua connessione termina su un server che richiede questa procedura. Quando rileva tale stato, esegue il comando corrispondente con la tua password salvata al posto tuo, senza bisogno di digitare nulla. Questo è l'intero meccanismo. Non ci sono manipolazioni di pacchetti, protocolli nascosti, o nulla che il server non accetterebbe già da un utente che preme invio dopo averlo digitato.

## Cosa tocca rispetto a cosa non tocca

| Area | Influenzato da AutoAuthenticate? |
|---|---|
| Comando chat di login/registrazione | Sì, inviato automaticamente all'ingresso |
| Combattimento (portata, tempismo dei colpi, mira) | No |
| Movimento (velocità, salti, percorso) | No |
| Qualsiasi azione di gioco registrata dal server | No |
| Archiviazione password | Locale al client e alla macchina |

L'ultima riga è la vera e propria "interfaccia" di impostazioni: una password salvata per server, conservata localmente per poterla riutilizzare al successivo ingresso. Tutto il resto è solo automazione di un messaggio di chat che già avevi il permesso di inviare.

I server che si affidano a plugin di autenticazione richiedono lo stesso comando ad ogni riconnessione, e se ti disconnetti e riconnetti spesso, digitare una password ogni volta diventa un vero fastidio durante una sessione. AutoAuthenticate elimina completamente questo fastidio senza intaccare nulla che un server o un anticheat si preoccupa di misurare.
