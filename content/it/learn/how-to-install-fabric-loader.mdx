<KeyTakeaways>

- Il Fabric Loader si installa con un unico strumento: l'installer ufficiale di Fabric.
- Aggiunge un nuovo profilo di lancio e non modifica mai il tuo gioco vanilla.
- La cartella `mods` compare dopo aver lanciato il profilo Fabric una volta.
- Abbina il loader alla versione esatta di Minecraft che intendi utilizzare.

</KeyTakeaways>

L'installazione del loader [Fabric](https://fabricmc.net/) richiede un solo strumento e pochi clic. Scarica il file di installazione ufficiale di Fabric, eseguilo, seleziona la versione di Minecraft desiderata e lascialo creare un nuovo profilo di avvio. Apri il tuo launcher, seleziona il profilo Fabric e avvia il gioco una volta. Il loader è ora attivo e pronto per i mod.

</Callout>

<Steps>

<Step number={1} title="Scarica il programma di installazione">Scarica l'installer ufficiale di [Fabric](https://fabricmc.net/) dal sito web del progetto Fabric. È un piccolo programma Java</Step>

<Step number={2} title="Eseguilo">
Fai doppio clic sul file jar, oppure eseguilo da un terminale se il doppio clic non fa nulla.
</Step>

Avvia il gioco su quel profilo in modo che il loader configuri i suoi file e una cartella `mods`.
</Step>

## Cosa fa effettivamente l'installer

L'installer non applica patch al file jar del gioco. Aggiunge un profilo di avvio che carica il loader [Fabric](https://fabricmc.net/) prima che Minecraft si avvii. Tale loader legge la cartella `mods` all'avvio e carica ciascun mod. Poiché il profilo si trova accanto a quello vanilla, il tuo gioco non modificato rimane intatto e puoi tornare indietro in qualsiasi momento.

## Installer vs supporto del launcher

Hai due modi per installare il loader. Scegli quello più adatto alla tua configurazione.

| Metodo | Quando usarlo | Note |
|--------|----------------|-------|
| Installer ufficiale di Fabric | Predefinito per il launcher standard | Scrive un profilo, scarica il loader, fatto in pochi minuti |
| Launcher con supporto Fabric integrato | Utilizzi già un launcher di terze parti | Il launcher gestisce l'abbinamento delle versioni per te |

Entrambi terminano allo stesso modo: un profilo Fabric che selezioni prima di avviare.

## Dove vengono salvati i file

| Sistema operativo | Percorso |
| :--- | :--- |
| Windows | `%appdata%\.minecraft` |
| macOS | `~/Library/Application Support/minecraft` |
| Linux | `~/.minecraft` |

La cartella `mods` appare all'interno di tale directory dopo il primo avvio di Fabric. Inserisci lì i file jar dei mod una volta che esiste.

## Quando il loader non si installerà

<Checklist>
<ChecklistItem>L'installatore non si apre: potrebbe mancare Java oppure il jar si apre con il programma sbagliato. Installare una versione recente di Java runtime e riprovare.</ChecklistItem>
<ChecklistItem>Nessun profilo Fabric nel launcher: aggiornare o riavviare il launcher, quindi cercare il nuovo profilo nel menu a tendina delle versioni.</ChecklistItem>
<ChecklistItem>Il gioco si blocca dopo aver aggiunto mod: il loader è a posto, ma una mod è indirizzata a una versione diversa di Minecraft. Assicurarsi che ogni mod sia compatibile con la versione installata.</ChecklistItem>
<ChecklistItem>Versione di Minecraft errata: rieseguire l'installatore e selezionare la versione corretta. Potrebbe scrivere più di un profilo.</ChecklistItem>
</Checklist>
