<KeyTakeaways>

- Fabric è un loader leggero e con aggiornamenti rapidi, preferito dai mod di performance e utilità.
- Forge è il loader più datato e pesante, con il catalogo più ampio di mod di contenuti di grandi dimensioni.
- Scegli Fabric per il supporto delle versioni iniziali e un'impronta ridotta; scegli Forge per modpack di grandi dimensioni.
- Una mod è indirizzata a un loader o all'altro, quindi abbina le tue mod al loader che utilizzi.

</KeyTakeaways>

Fabric è un mod loader leggero e veloce che supporta le nuove versioni di Minecraft nel giro di pochi giorni ed è preferito dai mod di prestazioni e utilità. Forge è il loader più datato e pesante, con il catalogo più ampio di mod di contenuti di grandi dimensioni. Scegliete Fabric per velocità e supporto delle versioni più recenti; scegliete Forge per modpack di grandi dimensioni e mod di contenuti che vengono rilasciate solo su di esso.

## Cosa sono l'uno e l'altro

Fabric è un mod loader modulare costruito attorno a un piccolo core più l'API Fabric, versionata separatamente ([Fabric API](/it/learn/what-is-the-fabric-api)). È noto per gli aggiornamenti rapidi alle nuove versioni di Minecraft e per un modello di patching basato su mixin.

Forge è il loader consolidato che ha alimentato la maggior parte dei grandi modpack per anni. Include una vasta API propria, che rende più semplici da sviluppare i mod con contenuti pesanti, ma rallenta l'aggiornamento del loader ad ogni versione.

## Come differiscono

| | Fabric | Forge |
| :--- | :--- | :--- |
| Peso | Core leggero + API Fabric | API integrata pesante e ampia |
| Supporto nuove versioni | Giorni dopo un rilascio | Settimane o mesi |
| Ottimo per | Mod di performance + utility | Mod con contenuti pesanti + modpack |
| Modello di patching | Mixin | Coremods e i propri hook |
| Compatibilità tra mod | API per mod | API condivisa ampia |

## Quale scegliere

- Scegli Fabric se vuoi le ultime versioni di Minecraft in anticipo, eseguire mod di performance o desideri un'impronta minima.
- Scegli Forge se i tuoi mod o modpack preferiti sono solo per Forge, o se vuoi una grande API condivisa su cui sviluppare.
- Molti giocatori ne mantengono entrambi, un profilo per loader, dato che la maggior parte dei mod è mirata a uno o all'altro.

## FAQ

<FAQList>

<FAQItem question="Fabric è più veloce di Forge?">
Il loader stesso è più leggero e le mod per le prestazioni più popolari che lo accompagnano (Sodium, Lithium) funzionano su Fabric. La velocità effettiva dipende dalle mod che installi, non solo dal loader.
</FAQItem>

<FAQItem question="Posso usare mod di Forge su Fabric?">
Non direttamente. Una mod creata per un loader non funziona sull'altro senza un livello di compatibilità, quindi verifica quale loader è richiesto da una mod prima di installarla.
</FAQItem>

<FAQItem question="Quale viene aggiornato più velocemente alle nuove versioni?">
Fabric è solitamente pronto entro pochi giorni dall'uscita di una versione di Minecraft. Forge di solito richiede più tempo a causa della sua maggiore superficie API.
</FAQItem>

</FAQList>
