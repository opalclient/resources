<KeyTakeaways>

- L'installazione di un mod Fabric richiede tre passaggi: il loader, l'API Fabric e poi il file jar del mod.
- Ogni componente (loader, API, mod) deve essere indirizzato alla stessa versione di Minecraft.
- I mod si trovano nella cartella "mods" all'interno della directory di Minecraft.
- La maggior parte dei fallimenti nell'installazione sono dovuti a una versione non corrispondente o alla mancanza dell'API Fabric.

</KeyTakeaways>

L'installazione di un mod per [Fabric](https://fabricmc.net/) richiede tre passaggi: installare il loader di Fabric per la tua esatta versione di Minecraft, inserire l'API di Fabric nella cartella mods, quindi trascinare il file jar del mod accanto ad esso. Avvia il profilo Fabric e il mod verrà caricato. La maggior parte dei problemi di installazione derivano da una versione non corrispondente, quindi assicurati che ogni componente sia compatibile con la stessa versione di Minecraft.

## I tre passaggi

<Steps>

<Step number={1} title="Installa il Fabric loader">
Installa il loader [Fabric](https://fabricmc.net/) per la tua versione di Minecraft, utilizzando l'installer ufficiale di Fabric o un launcher che supporta Fabric. Consulta [come installare il Fabric loader](/it/learn/how-to-install-fabric-loader) per la guida completa.
</Step>

<Step number={2} title="Aggiungi la Fabric API">Inserisci il file jar di [Fabric API](/it/learn/what-is-the-fabric-api) nella cartella delle mod. La maggior parte delle mod ne dipende</Step>

<Step number={3} title="Aggiungi la mod">Metti il file jar della mod nella stessa cartella mods, accanto a Fabric API</Step>

</Steps>

Avvia quindi il profilo Fabric e la mod si caricherà all'avvio. Se non compare, controlla l'elenco di risoluzione dei problemi qui sotto.

## Dove si trova la cartella mods

| Sistema operativo | Percorso |
| :--- | :--- |
| Windows | `%appdata%\.minecraft\mods` |
| macOS | `~/Library/Application Support/minecraft/mods` |
| Linux | `~/.minecraft/mods` |

Se la cartella `mods` non esiste ancora, esegui il profilo Fabric una volta e verrà creata automaticamente.

## Quando una mod non si carica

<Checklist>
<ChecklistItem>Il mod non compare: il file jar si trova nella cartella sbagliata oppure hai avviato un profilo non di Fabric.</ChecklistItem>
<ChecklistItem>Crash all'avvio con un errore di classe mancante: manca l'API di Fabric o è presente una versione errata.</ChecklistItem>
<ChecklistItem>Incompatibilità di versione: il loader, l'API di Fabric e il mod devono tutti puntare alla stessa versione di Minecraft.</ChecklistItem>
</Checklist>
