<KeyTakeaways>

- Fabric ist ein leichter, schnell aktualisierter Loader, der von Performance- und Utility-Mods bevorzugt wird.
- Forge ist der ältere, schwerere Loader mit dem umfangreichsten Katalog an umfangreichen Content-Mods.
- Wähle Fabric für die Unterstützung früherer Versionen und einen geringen Ressourcenverbrauch; wähle Forge für große Modpacks.
- Ein Mod richtet sich an einen der beiden Loader, also passe deine Mods dem Loader an, den du verwendest.

</KeyTakeaways>

Fabric ist ein schlanker, schneller Mod-Loader, der neue Minecraft-Versionen innerhalb weniger Tage unterstützt und bei Performance- und Utility-Mods bevorzugt wird. Forge ist der ältere, schwerere Loader mit dem umfangreichsten Katalog an umfangreichen Content-Mods. Wähle Fabric für Geschwindigkeit und frühe Versionsunterstützung; wähle Forge für große Modpacks und Content-Mods, die nur dort verfügbar sind.

## Was sie ausmacht

Fabric ist ein modulares Mod-Laden-System, das auf einem kleinen Kern plus der separat versionierten [Fabric API](/de/learn/what-is-the-fabric-api) basiert. Es ist bekannt dafür, schnell auf neue Minecraft-Versionen zu aktualisieren und ein sauberes, mixin-basiertes Patching-Modell zu verwenden.

Forge ist das seit langem etablierte Ladeprogramm, das die meisten großen Modpacks über Jahre hinweg unterstützt hat. Es verfügt über eine umfangreiche eigene API-Oberfläche, was die Entwicklung von umfangreichen Inhaltsmods erleichtert, aber das Ladeprogramm langsamer bei der Aktualisierung jeder Version macht.

## Worin sie sich unterscheiden

| | Fabric | Forge |
| :--- | :--- | :--- |
| Gewicht | Leichtgewichtiger Kern + Fabric API | Schwer, große integrierte API |
| Unterstützung für neue Versionen | Tage nach einer Veröffentlichung | Wochen bis Monate |
| Stärken | Leistung + Utility-Mods | Umfangreiche Inhaltsmods + Modpacks |
| Patching-Modell | Mixins | Coremods und eigene Hooks |
| Cross-Mod-Kompatibilität | API pro Mod | Breite, gemeinsame API |

## Welche Wahl treffen?

- Wählen Sie Fabric, wenn Sie die neueste Minecraft-Version frühzeitig nutzen möchten, Leistungsmods ausführen oder einen minimalen Fußabdruck wünschen.
- Wählen Sie Forge, wenn Ihre Lieblings-Inhaltsmods oder Modpacks nur für Forge verfügbar sind oder Sie eine große, gemeinsame API zum Bauen wünschen.
- Viele Spieler behalten beide, ein Profil pro Ladeprogramm, da die meisten Mods auf eines der beiden abzielen.

## FAQ

<FAQList>

<FAQItem question="Ist Fabric schneller als Forge?">
Der Loader selbst ist leichter, und die beliebten Performance-Mods, die damit kombiniert werden (Sodium, Lithium), laufen auf Fabric. Die Rohgeschwindigkeit hängt von den installierten Mods ab, nicht vom Loader allein.
</FAQItem>

<FAQItem question="Kann ich Forge-Mods auf Fabric ausführen?">
Nicht direkt. Ein Mod, der für einen Loader entwickelt wurde, lässt sich nicht ohne eine Kompatibilitätsschicht auf dem anderen Loader laden. Überprüfen Sie daher, für welchen Loader ein Mod bestimmt ist, bevor Sie ihn installieren.
</FAQItem>

<FAQItem question="Welche Version wird schneller aktualisiert?">
Fabric ist typischerweise innerhalb weniger Tage nach einer Minecraft-Veröffentlichung fertig. Forge dauert meist länger, da es eine größere API-Schnittstelle hat.
</FAQItem>

</FAQList>
