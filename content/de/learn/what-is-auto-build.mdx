<KeyTakeaways>

- Auto Build ist ein Weltmodul, das ein gespeichertes Schema automatisch, Block für Block, konstruiert.
- Es folgt einem festen Layout und platziert den passenden Block aus deinem Inventar an jeder Position.
- Es ist ein Baufahrzeug, kein Kampffahrzeug: Reichweite, Zielgenauigkeit und Trefferzeitpunkt bleiben unbeeinflusst.
- Der Sinn ist, schneller als ein Mensch bauen zu können, und das kann auf Survival-Servern auffallen.

</KeyTakeaways>

Auto Build ist ein Weltmodul, das automatisch eine vordefinierte Struktur oder ein Schema konstruiert und dabei ein gespeichertes, blockweise aufgebautes Layout durchläuft, ohne dass der Spieler etwas manuell platzieren muss. Laden Sie eine Struktur, und das Modul platziert jeden einzelnen Block selbst und zieht dabei die passenden Blöcke aus Ihrem Inventar. Es ist in erster Linie ein Werkzeug zum Bauen. Es ändert nichts an Ihrer Kampfweise oder Fortbewegung, sondern nur an der Geschwindigkeit, mit der eine Mauer errichtet wird.

## Funktionsweise

Auto Build beginnt mit einer vordefinierten Struktur oder einem Schema: einem gespeicherten Layout, das genau festlegt, welcher Block wo platziert werden soll. Das Modul arbeitet dieses Layout Position für Position ab und platziert den korrekten Block an jeder Stelle aus den passenden Blöcken, die sich in Ihrem Inventar befinden. Sie müssen nicht zielen, klicken oder sich selbst an den Ort bewegen. Die Platzierungssequenz läuft eigenständig ab.

Da die Struktur im Voraus festgelegt ist, muss das Modul keine Entscheidungen mehr treffen. Es führt eine bekannte Liste von Koordinaten und Blocktypen aus, und das kann es schneller und konsistenter als eine Person, die jeden einzelnen Block manuell anklickt, erledigen.

Wurst, der Client, der dieses Modul dokumentiert, ordnet es unter Bauen und Weltinteraktion ein, anstatt unter Kampf. Das ist der ganze Reiz: eine Bau-Abkürzung für Türme, Pyramiden und andere Mehrblock-Layouts, die ewig dauern, wenn man sie einzeln von Hand platziert, wobei ein Fehler auf halbem Weg normalerweise bedeutet, dass man ihn selbst beheben muss. Auto Build verlegt nie einen Block und verliert nie den Überblick, wo er sich im Layout befindet.

## Ist es sicher zu verwenden?

Auto Build ist kein Stealth-PvP-Modul wie KillAura oder Scaffold. Es greift weder Reichweite, Zielgenauigkeit noch Trefferzeitpunkte an, und die meisten Leute nutzen es zum Bauen, nicht zum Kämpfen, daher hat es nicht das gleiche Erkennungsprofil wie ein Combat-Cheat.

Es ist zwar automatisierte Blockplatzierung, und das allein kann schon ausreichen. Ein Server, der die Platzierungsrate oder das Muster aus Gründen der Verhinderung von Griefing oder Betrug überwacht, kann eine Struktur, die schneller hochgeht, als ein Mensch sie bauen könnte, als verdächtig einstufen, insbesondere im Survival-Mehrspielermodus. In Singleplayer- oder kreativitätsorientierten Umgebungen, in denen Bautechniken bereits erwartet werden, verschwindet dieses Risiko größtenteils.

## Häufig gestellte Fragen

<FAQList>

<FAQItem question="Was baut der Auto-Build-Modus eigentlich?">
Was auch immer eine vordefinierte Struktur oder ein Schema geladen wurde. Das Modul entwirft nichts von selbst. Es folgt einem gespeicherten Layout und platziert Blöcke entsprechend, Position für Position.
</FAQItem>

<FAQItem question="Hilft Auto Build im Kampf?">
Nein. Es ist ein Modul für Welt- und Baukunst, nicht für den Kampf. Es hat keinen Einfluss auf Reichweite, Zielgenauigkeit oder Angriffszeitpunkt, und das Ausführen ändert nichts daran, wie ein Kampf abläuft.
</FAQItem>

<FAQItem question="Welche Client-Dokumente werden von Auto Build erstellt?">
Wurst ist der Client, in dem dieses Modul dokumentiert ist und unter den Werkzeugen zur Erstellung und Interaktion mit der Welt geführt wird, anstatt unter dem Kampfteil des Clients.
</FAQItem>

<FAQItem question="Kann Auto Build dazu führen, dass man gesperrt wird?">
Es ist unwahrscheinlich, dass es die gleichen Erkennungen wie ein Combat-Cheat auslöst, aber automatisches Blockieren kann dennoch von Servern markiert werden, die auf ungewöhnlich schnelle Bauaktivitäten achten, insbesondere in Survival-Mehrspieler-Welten.
</FAQItem>
