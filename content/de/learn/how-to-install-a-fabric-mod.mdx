<KeyTakeaways>

- Die Installation eines Fabric-Mods besteht aus drei Schritten: Loader, Fabric API und dann die Mod-Datei.
- Jedes Element (Loader, API, Mod) muss für die gleiche Minecraft-Version sein.
- Mods befinden sich im "mods"-Ordner innerhalb deines Minecraft-Verzeichnisses.
- Die meisten Installationsfehler sind auf eine falsche Version oder eine fehlende Fabric API zurückzuführen.

</KeyTakeaways>

Die Installation eines [Fabric](https://fabricmc.net/) Mods erfordert drei Schritte: Installieren Sie den Fabric Loader für Ihre genaue Minecraft-Version, legen Sie die Fabric API in Ihren Mods-Ordner und legen Sie dann die Jar-Datei des Mods daneben ab. Starten Sie das Fabric Profil, und der Mod wird geladen. Die meisten Installationsprobleme entstehen durch eine Versionsunterschied, daher sollten Sie alle Komponenten auf dieselbe Minecraft-Version abstimmen.

## Die drei Schritte

<Steps>

<Step number={1} title="Installiere den Fabric Loader">Installieren Sie den [Fabric](https://fabricmc.net/) Loader für Ihre Minecraft-Version mit dem offiziellen Fabric-Installer oder einem Launcher, der Fabric unterstützt. Sehen Sie sich [die vollständige Anleitung zur Installation des Fabric Loaders](/de/learn/how-to-install-fabric-loader) an</Step>

<Step number={2} title="Füge die Fabric API hinzu">Lege die [Fabric API](/de/learn/what-is-the-fabric-api)-Jar-Datei in deinen Mods-Ordner. Die meisten Mods sind davon abhängig</Step>

<Step number={3} title="Füge den Mod hinzu">Lege die Jar-Datei des Mods in denselben Mods-Ordner, neben der Fabric API</Step>

</Steps>

Dann starten Sie das Fabric-Profil und der Mod wird beim Start geladen. Sollte er nicht erscheinen, überprüfen Sie die untenstehende Fehlerbehebung.

## Wo sich der Mods-Ordner befindet

| Betriebssystem | Pfad |
| :--- | :--- |
| Windows | `%appdata%\.minecraft\mods` |
| macOS | `~/Library/Application Support/minecraft/mods` |
| Linux | `~/.minecraft/mods` |

Wenn der `mods`-Ordner noch nicht existiert, starten Sie das Fabric-Profil einmal und er wird für Sie erstellt.

## Wenn ein Mod nicht geladen wird

<Checklist>
<ChecklistItem>Das Mod erscheint nicht: Die Jar-Datei befindet sich im falschen Ordner oder Sie haben ein nicht-Fabric-Profil gestartet.</ChecklistItem>
<ChecklistItem>Absturz beim Start mit einem fehlenden Klassenfehler: Die Fabric API fehlt oder ist in der falschen Version.</ChecklistItem>
<ChecklistItem>Versionsunterschied: Der Loader, die Fabric API und das Mod müssen alle auf dieselbe Minecraft-Version abzielen.</ChecklistItem>
</Checklist>
