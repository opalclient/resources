<KeyTakeaways>

- Fabric ist ein leichtgewichtiger Mod-Loader; installieren Sie zuerst den Fabric Loader und legen Sie anschließend die Mod-JAR-Dateien in den Mods-Ordner.
- Für Minecraft 1.21 benötigen Sie Java 21 oder neuer, und Minecraft sollte mindestens einmal gestartet worden sein, bevor Sie installieren.
- Mod-JAR-Dateien finden Sie unter %appdata%\.minecraft\mods auf Windows (mit macOS- und Linux-Äquivalenten).
- Opal verwendet ein Loader-System: der Opal Launcher übernimmt Fabric, Authentifizierung, Installation und Updates für Sie.

</KeyTakeaways>

Fabric ist ein leichtgewichtiger [Mod-Loader](/de/learn/what-is-a-minecraft-mod-loader) für Minecraft. Er ist schnell, wird aktiv gepflegt und unterstützt moderne Minecraft-Versionen. Viele Utility-Clients, darunter Opal, laufen auf Fabric. Dieser Leitfaden führt Sie durch die Installation von Fabric Loader, das Hinzufügen von Mods und die Einrichtung von Opal
## Voraussetzungen
Bevor Sie beginnen, stellen Sie sicher, dass Sie Folgendes haben:

- **Minecraft Java Edition** installiert und mindestens einmal gestartet
- **Java 21** oder neuer installiert (erforderlich für MC 1.21+)
- Eine funktionierende Internetverbindung
## Installiere Fabric Loader

<Steps>

<Step title="Laden Sie den Fabric-Installer herunter">

Besuchen Sie die offizielle Fabric-Website unter **fabricmc.net**, klicken Sie auf **Download**, wählen Sie die Option **Installer** aus und laden Sie den Universal JAR-Installer herunter

</Step>

<Step title="Führen Sie den Installer aus und bestätigen Sie die Einstellungen">

Führen Sie die heruntergeladene Fabric-Installer-JAR-Datei aus und stellen Sie anschließend sicher, dass diese Einstellungen korrekt sind:

- Der Reiter **Client** ist ausgewählt
- **Minecraft-Version:** 1.21.10 (oder Ihre Zielversion)
- **Loader-Version:** Neueste stabile Version (0.17.x empfohlen)
- **Installationsort:** Ihr Standard-.minecraft-Verzeichnis

Klicken Sie auf **Installieren**. Sie sollten eine Erfolgsmeldung sehen

</Step>

<Step title="Überprüfen Sie die Installation">

Öffnen Sie den Minecraft Launcher, klicken Sie auf den Reiter **Installationen** und suchen Sie nach einem neuen Profil namens **fabric-loader-1.21.10** (oder ähnlich). Wählen Sie dieses Profil aus und klicken Sie auf **Play**. Wenn Minecraft mit dem Fabric-Logo auf dem Titelscreen startet, sind Sie bereit

</Step>

</Steps>

## Mods installieren

Fabric-Mods kommen in deinen `mods`-Ordner:

- **Windows:** `%appdata%\.minecraft\mods`
- **macOS:** `~/Library/Application Support/minecraft/mods`
- **Linux:** `~/.minecraft/mods`

Falls der `mods`-Ordner nicht existiert, erstelle ihn. Um einen Mod zu installieren:

<Steps>

<Step title="Lade die Mod-JAR herunter">

Lade die `.jar`-Datei des Mods von seiner Quelle herunter

</Step>

<Step title="Lege es in den Mods-Ordner">

Verschieben Sie die `.jar`-Datei in den `mods`-Ordner für Ihre Plattform

</Step>

<Step title="Starten Sie mit dem Profil Fabric">

Starte Minecraft mit dem Fabric-Profil, nicht mit der Vanilla-Version

</Step>

</Steps>

## Opal Client installieren

Opal verwendet ein **Ladesystem**: Sie legen den kompletten Client nicht in den Mods-Ordner. Stattdessen:

<Steps>

<Step title="Kaufen Sie Opal">

Kaufen Sie Opal vom Armaturenbrett ($8.99 lebenslang)

</Step>

<Step title="Laden Sie den Opal Launcher herunter">

Laden Sie den **Opal Launcher** von der Download-Seite herunter. Er übernimmt Authentifizierung, Installation und Updates automatisch

</Step>

<Step title="Oder verwende die Loader-JAR">

Alternativ können Sie die Opal Loader JAR-Datei herunterladen und in Ihren `mods`-Ordner legen. Sie authentifiziert sich und lädt den Client zur Laufzeit herunter

</Step>

</Steps>

Der Launcher ist der empfohlene Ansatz: Er verwaltet alles für Sie.

## Fehlerbehebung

### "Fabric wird im Launcher nicht angezeigt"
Stellen Sie sicher, dass Sie Fabric für die richtige Minecraft-Version installiert haben. Führen Sie den Installer bei Bedarf erneut aus.

### "Spiel stürzt beim Start ab"
- Überprüfen Sie, ob **Java 21** installiert ist (nicht Java 8 oder 17)
- Entfernen Sie inkompatible Mods aus dem `mods`-Ordner
- Stellen Sie sicher, dass der Fabric Loader auf dem neuesten Stand ist

### "Mods werden nicht geladen"
- Vergewissern Sie sich, dass die `.jar`-Dateien im richtigen `mods`-Ordner liegen
- Stellen Sie sicher, dass Sie mit dem **Fabric-Profil** und nicht mit Vanilla starten
- Überprüfen Sie, ob die Mod-Version mit Ihrer Minecraft-Version übereinstimmt

### "Opal kann sich nicht authentifizieren"
- Stellen Sie sicher, dass Sie eine aktive Internetverbindung haben
- Vergewissern Sie sich, dass Ihr Lizenzschlüssel im Dashboard aktiviert ist
- Überprüfen Sie, ob Ihre HWID sich geändert hat (Zurücksetzen alle 30 Tage im Dashboard verfügbar)
- Benutzer in Regionen mit Internetbeschränkungen benötigen möglicherweise einen Proxy: Details finden Sie auf der Kaufseite
