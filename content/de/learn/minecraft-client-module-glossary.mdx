    a: Sichere Module sind kosmetische oder komfortorientierte Änderungen, die nur Ihren eigenen Client betreffen und Anticheats nichts anmerken lassen. <KeyTakeaways>

- Opal versendet 70 Module über sechs Kategorien: Kampf, Welt, Bewegung, Hilfsmittel, Visuelles und Sonstiges.
- 21 davon sind sichere, kosmetische Änderungen, die nur Ihren eigenen Client betreffen. 49 sind Funktionen, die auf eigene Gefahr verwendet werden können und das Gameplay auf eine Weise verändern, die von Anti-Cheat-Systemen erkannt werden kann.
- Jedes unten aufgeführte Modul verlinkt zu einer vollständigen Erläuterung, wie es funktioniert und warum es erkannt wird oder eben nicht.
- Diese Seite ist die Karte. Beginnen Sie hier und tauchen Sie dann tief in das ein, was Sie tatsächlich tun möchten.

</KeyTakeaways>

Ein Modul ist eine einzelne Funktion, die du in Opal aktivieren kannst, von etwas so Unauffälligem wie einem Discord-Status bis hin zu etwas so Auffälligem wie einem Aim-Bot. Einige bieten reine Bequemlichkeit. Andere führen zu einer Sperre auf einem Server, der aufpasst. Zu wissen, welche Funktion welche ist, ist wichtiger, als zu wissen, dass sie überhaupt existieren, weshalb diese Seite alle 70 Module nach Kategorie sortiert und jedes mit „sicher“ oder „auf eigene Gefahr“ kennzeichnet, bevor du überhaupt einen Schalter betätigst.

## Sicher vs. auf eigene Gefahr, in einer Tabelle

| | Sicher | Auf eigene Gefahr |
|---|---|---|
| **Was es beeinflusst** | Nur die Darstellung oder Bequemlichkeit deines eigenen Clients | Bewegung, Kampf oder Timing, das der Server beobachten kann |
| **Anticheat-Erkennung** | Keine, es werden keine ungewöhnlichen Daten an den Server gesendet | Real, genau das, worauf Anticheats ausgelegt sind |
| **Beispiele** | Vollständige Helligkeit, Klick-GUI, Discord RPC, Streamer-Modus | Kill Aura, Gerüst, Geschwindigkeit, Reichweite |
| **Wo es einwandfrei läuft** | Jeder Server, jederzeit | Hängt vollständig davon ab, wie streng dieser Server seine Regeln durchsetzt |

Das Label ist keine Garantie, sondern ein Ausgangspunkt. Lies die spezielle Seite für jedes Modul, das du tatsächlich verwenden möchtest, bevor du es aktivierst.

## Wie man Module richtig auswählt

Schalte nicht alles ein. Beginne mit dem, was das spezifische Problem löst, das du hast: Volle Helligkeit, wenn Höhlen deine Zeit verschwenden, Automatische Werkzeuge und Automatische Rüstung, wenn die Inventarverwaltung der langweilige Teil deines Tages ist, nur Kill Aura, wenn du bereits das Bannrisiko akzeptiert hast, das damit verbunden ist. Staple sichere Module bedenkenlos. Behandle jedes Modul mit Risiken als eine Entscheidung, nicht als Standardeinstellung.

<Callout variant="tip" title="Wo Sie anfangen können">
Neu bei Opal? Die sichere Spalte oben ist ein wirklich guter Ort, um Ihre erste Sitzung zu verbringen: Klicken Sie auf GUI, um zu sehen, was verfügbar ist, Full Bright und Streamer-Modus, wenn Sie streamen, und den Bestandsmanager, wenn Sie von Unordnung genervt sind. Keiner dieser Punkte birgt ein Risiko für Ihr Konto.
</Callout>

<FAQItem question="Was ist der Unterschied zwischen einem sicheren Modul und einem Modul mit Eigenverantwortung?">
