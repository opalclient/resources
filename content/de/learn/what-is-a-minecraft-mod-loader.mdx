<KeyTakeaways>

- Ein Mod-Loader ist die Schicht, die es Minecraft ermöglicht, Mods auszuführen; die Vanilla-Version bietet keine Mod-Unterstützung.
- Er patcht das Spiel beim Start und gibt jedem Mod eine Möglichkeit, sich in den Code einzuhaken.
- Du installierst ihn einmal für deine Minecraft-Version und legst dann die Mod-JAR-Dateien in den Mods-Ordner.
- Die Hauptoptionen sind Fabric, Forge, NeoForge und Quilt; wähle die aus, die von deinen Mods verwendet wird.

</KeyTakeaways>

Ein Mod-Loader ist die Schicht, die es Minecraft ermöglicht, Mods auszuführen. Vanilla Minecraft hat keine Mod-Unterstützung, daher patcht ein Loader das Spiel beim Start und gibt Mods die Möglichkeit, sich einzuhängen. Du installierst den Loader einmal für deine Minecraft-Version und legst dann die Mod-JAR-Dateien in den Mods-Ordner, wo sie beim Start geladen werden.

## Warum du eines brauchst

Mods verändern, wie das Spiel funktioniert, was bedeutet, dass sie innerhalb des Spielcodes ausgeführt werden müssen. Der Loader macht das möglich. Ohne ihn ist eine Mod-Datei nur eine Datei, die das Spiel ignoriert.

## Wie es funktioniert

Beim Start fügt der Loader sich vor dem Spielstart ein, liest den Mods-Ordner und verbindet jeden Mod mit den richtigen Hooks. [Fabric](https://fabricmc.net/) verwendet Mixins, einen Ansatz zur gezielten Bytecode-Patchung. Forge verwendet sein eigenes, größeres Hook-System. Auf jeden Fall läuft das Spiel mit den eingestickten Mods.

## Die wichtigsten Loader

| Loader | Bekannt für |
| :--- | :--- |
| Fabric | Leichtgewichtig, schnell aktualisierbar, Performance- und Utility-Mods |
| Forge | Umfangreiche Content-Mods und etablierte Modpacks |
| NeoForge | Eine von der Community geprägte Fork von Forge |
| Quilt | Eine Fork von Fabric mit zusätzlichen Funktionen |

## Einen auswählen

Verwende den Loader, auf den deine Mods abzielen. Die meisten Mods werden für einen Loader und nicht für beide ausgeliefert, sodass die Mods, die du verwenden möchtest, diese in der Regel für dich bestimmen. Für eine tiefere Aufschlüsselung siehe [Fabric vs Forge](/de/learn/fabric-vs-forge).
