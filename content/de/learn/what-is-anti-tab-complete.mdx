<KeyTakeaways>
- AntiTabComplete ist ein Minecraft-Modul, das Tab-Complete-Anfragen blockiert, die einige Server nutzen, um unauffällig die Client-Daten eines Spielers zu profilieren.
- Es fängt die Client-Seite des Minecraft-Tab-Completion-Austauschs ab, den Hintergrundverkehr, der durch das Drücken der Tab-Taste ausgelöst wird.
- Es ist eine Privatsphäre-Verbesserung, keine Wettbewerbsverbesserung; es greift niemals Bewegung oder Kampf an.
- Tenacity ist der Client, der mit diesem Modul dokumentiert ist.
</KeyTakeaways>

AntiTabComplete ist ein Modul, das verhindert, dass ein Server die Tab-Completion-Funktion von Minecraft nutzt, um unbemerkt Informationen über den Client eines Spielers abzurufen. Einige Server-Plugins senden Tab-Completion-Anfragen nicht, um die Eingabe zu erleichtern, sondern als unauffälligen Kanal zur passiven Datenerfassung. AntiTabComplete schließt diesen Kanal von der Client-Seite aus, sodass diese Anfragen einfach keine Antwort erhalten.

## Funktionsweise

Die Tab-Vervollständigung ist normales Minecraft-Verhalten. Beginnen Sie mit der Eingabe eines Befehls, drücken Sie die Tab-Taste und der Client und Server tauschen Nachrichten im Hintergrund aus, damit das Spiel gültige Vervollständigungen vorschlagen kann. Dieser Austausch wird dem Spieler nicht angezeigt, was ihn genau für etwas anderes als die automatische Vervollständigung wiederverwendbar macht. Ein Plugin kann diese Anfragen an einen Client senden, um ihn zu untersuchen und alles zurückgelesene zu lesen, ohne dass eine einzige Zeile davon im Chat erscheint.

AntiTabComplete füllt diese Lücke. Es fängt oder ignoriert die Anfragen, bevor der Client sie beantworten kann, sodass ein Plugin, das sich auf den Tab-Completion-Kanal verlässt, nichts Nützliches zurückbekommt. Normale Chat- und Befehlseingaben sehen für den Spieler immer gleich aus. Nichts ändert sich sichtbar, und das ist der Sinn der Sache.

## Eine Privatsphäre-Verbesserung, kein Vorteil im Spiel

Hier weicht AntiTabComplete von dem ab, was die meisten Menschen meinen, wenn sie von einem „Minecraft-Cheat“ sprechen. Ein Modul wie [Reach](/de/learn/what-is-reach) oder KillAura verändert, was das Spiel physisch erlaubt, und Anticheats existieren speziell, um die Bewegungs- und Timing-Fingerabdrücke zu erkennen, die dadurch hinterlassen werden. AntiTabComplete greift weder die Bewegung noch den Kampf an. Es verändert lediglich, was ein Server passiv über dich lernen kann, was es zu einem Informationsproblem und nicht zu einem Wettbewerbsproblem macht.

Das macht es auch risikofrei in der Anwendung. Es sendet keine gefälschten Daten und verändert nichts, was der Server über deinen Kampf oder deine Bewegung überprüfen kann; es unterlässt lediglich eine Antwort auf eine Anfrage, die der Spieler von vornherein nicht sehen sollte. Von der Serverseite aus sieht ein Client, der nicht auf eine passive Anfrage antwortet, einfach wie ein Client aus, dem das abgefragte Verhalten fehlt, und nicht wie ein Client, der gefälschte Pakete sendet. In derselben Kategorie wie andere Gegenüberwachungs-Tools: leise, defensiv und nicht das, was Anticheats tatsächlich erkennen sollen.
