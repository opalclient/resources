<KeyTakeaways>

- HWID (Hardware-ID) ist ein Fingerabdruck, der aus der Hardware Ihres Computers erstellt wird und dazu dient, eine Lizenz an ein Gerät zu binden.
- Kunden nutzen ihn, um Lizenzweitergabe zu verhindern: eine Lizenz entspricht einem Gerät.
- Umfangreiche Hardware-Austausche oder Betriebssystem-Neuinstallationen können Ihre HWID ändern; das Hinzufügen von RAM oder eines USB-Geräts hat in der Regel keine Auswirkungen.
- Opal ermöglicht es Ihnen, Ihre HWID einmal alle 30 Tage im Dashboard zurückzusetzen, mit Discord-Support für Notfälle.

</KeyTakeaways>

HWID steht für **Hardware-ID**, einen eindeutigen Identifikator, der aus den Hardwarekomponenten Ihres Computers generiert wird. Stellen Sie es sich als einen Fingerabdruck für Ihr Gerät vor. Bei der Aktivierung eines [Clients](/de/learn/what-is-a-minecraft-utility-client) wie Opal wird Ihre HWID gespeichert. Von diesem Zeitpunkt an kann nur dieses spezifische Gerät Ihre Lizenz nutzen.

## Warum Clients HWID verwenden

Die HWID-Lizenzierung löst ein einfaches Problem: **die Verhinderung von Lizenzweitergabe**.

Ohne HWID könnte eine Person eine Lizenz kaufen und sie mit Dutzenden von Freunden teilen. Das ist schlecht für Entwickler, die Einnahmen benötigen, um den Client zu warten und zu aktualisieren. HWID stellt sicher, dass eine Lizenz einem Gerät entspricht.

### Wie es generiert wird

Ihre HWID wird typischerweise aus einer Kombination von Folgendem abgeleitet:
- CPU-Identifikator
- Serielle Nummer des Motherboards
- Serielle Nummern der Festplattenlaufwerke
- MAC-Adresse
- Andere Hardware-Identifikatoren

Die genaue Kombination variiert je nach Client. Ziel ist ein stabiler Identifikator, der sich nicht ändert, es sei denn, Sie ändern Ihre Hardware erheblich.

## Wann sich Ihre HWID ändert

Ihre HWID kann sich ändern, wenn Sie:
- **Hauptkomponenten austauschen:** neues Motherboard, CPU oder Speichermedium
- **Ihr Betriebssystem neu installieren:** einige Identifikatoren werden zurückgesetzt
- **Zu einem anderen Computer wechseln:** offensichtlich eine völlig andere HWID
- **Bestimmte Systemtools verwenden:** Festplattenklonierung oder Hardware-Spoofing

Geringfügige Änderungen (Hinzufügen von RAM, Einstecken eines USB-Geräts) beeinflussen Ihre HWID in der Regel nicht.

## HWID-Zurücksetzungen

Da Hardware-Änderungen vorkommen (man rüsten seinen PC auf, wechselt zu einem Laptop oder installiert Windows neu), bieten die meisten Clients **HWID-Zurücksetzungen** an.

<Callout variant="info" title="Opal's HWID-Zurücksetzrichtlinie">
Sie können Ihre HWID über das Web-Dashboard unter Kontoeinstellungen zurücksetzen. Zurücksetzungen sind einmal alle 30 Tage verfügbar. Starten Sie nach dem Zurücksetzen Opal auf Ihrem neuen Gerät, um die neue HWID zu binden. Wenn Sie eine Notfall-Zurücksetzung vor dem Cooldown benötigen, kontaktieren Sie den Support über Discord.
</Callout>

### Häufige Reset-Szenarien

| Szenario | Reset erforderlich? |
|----------|---------------|
| Neuer Computer | Ja |
| Windows neu installiert | Vielleicht (hängt davon ab, was sich geändert hat) |
| Neues Mainboard/CPU | Ja |
| RAM hinzugefügt | Nein |
| Neue GPU | Normalerweise nein |
| Dual-Booting | Vielleicht (hängt vom Betriebssystem ab) |

## Tipps zur Verwaltung von HWID

- **Vermeiden Sie unnötige Resets:** Wenn Sie ein Hardware-Upgrade durchführen möchten, planen Sie den Reset für nach allen Änderungen.
- **Behalten Sie den Zugriff auf Ihr Dashboard:** HWID-Resets werden über das Web-Dashboard durchgeführt, stellen Sie also sicher, dass Ihre Kontodaten gespeichert sind.
- **Kontaktieren Sie den Support für Sonderfälle:** Wenn Sie die 30-Tage-Abkühlzeit erreicht haben, aber einen legitimen Bedarf haben, kann der Discord-Support helfen.

## HWID vs. andere Lizenzmodelle

| Modell | Funktionsweise | Vorteile | Nachteile |
|-------|-------------|------|------|
| **HWID** | An die Hardware gebunden | Einfach, kein Login im Spiel erforderlich | Hardware-Änderungen erfordern einen Reset |
| **Accountspezifisch** | An den Login gebunden | Funktioniert auf jedem Gerät | Zugangsdaten können geteilt oder gestohlen werden |
| **IP-basiert** | An die IP-Adresse gebunden | Keine Einrichtung erforderlich | VPNs und dynamische IPs stören es |
| **Abonnement** | Zeitlich begrenzt | Einnahmen für Entwickler | Laufende Kosten für Benutzer |

Opal verwendet HWID in Kombination mit Kontoverifizierung: Sie benötigen sowohl eine gültige Lizenz als auch die passende Hardware. Dies bietet eine starke Sicherheit bei gleichzeitig einfacher Benutzererfahrung.
