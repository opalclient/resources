Opal liefert eine GraalVM JavaScript-Engine, eine vollständige JS-Laufzeitumgebung, die innerhalb des Minecraft-Clients läuft. Sie schreiben Skripte, die mit den internen APIs von Opal interagieren, während das Spiel aktiv ist. Dies ist weder ein Makrorekorder noch ein Keybind-System. Es ist eine echte Programmierumgebung mit Zugriff auf das Ereignissystem des Clients, die Modulkonfigurationen und den Spielzustand.

<KeyTakeaways>

- Eine vollständige GraalVM JavaScript-Laufzeitumgebung, kein Makrorekorder.
- Reagiere auf Spieleereignisse, ändere Moduleinstellungen und zeichne deine eigene HUD.
- Skripte werden zur Laufzeit geladen, kein Neustart erforderlich, und laufen in einer Sandbox für Opals APIs.
- Die meisten anderen Clients stoppen bei einfachen Makros.

</KeyTakeaways>

## Warum GraalVM
GraalVM ist eine polyglotte Laufzeitumgebung von Oracle. Wir haben es aus konkreten Gründen ausgewählt:

- Geschwindigkeit: JIT-Kompilierung sorgt dafür, dass Skripte schnell genug sind, um jeden Takt auszuführen.
- JavaScript: Die meisten Leute kennen es bereits, sodass die Lernkurve kurz ist.
- Thread-Sicherheit: Die JS-zu-Java-Bindungen behandeln Nebenläufigkeit korrekt, anstatt Sie darüber stolpern zu lassen.
- Vertraute Muster: Standard-JavaScript-APIs verhalten sich so, wie Sie es erwarten
## Was du bauen kannst

### Benutzerdefinierte Automatisierung
Reagiere auf Ereignisse im Spiel:

```javascript
// Auto-respond to certain chat messages
on('chat', (event) => {
  if (event.message.includes('1v1?')) {
    sendChat('/duel accept');
  }
});
```
### Dynamische Modulkonfiguration
Ändere Moduleinstellungen basierend auf dem Spielzustand:

```javascript
// Increase reach when health is low
on('tick', () => {
  const health = player.getHealth();
  if (health < 8) {
    modules.get('Reach').setSetting('distance', 3.3);
  } else {
    modules.get('Reach').setSetting('distance', 3.1);
  }
});
```
### Benutzerdefinierte HUD-Elemente
Erstellen Sie Ihre eigenen Überlagerungen:

```javascript
// Display a combo counter on screen
let combo = 0;
on('attackEntity', () => { combo++; });
on('damage', () => { combo = 0; });
on('render2d', (ctx) => {
  ctx.drawText(`Combo: ${combo}`, 10, 10, 0xFFFFFF);
});
```
### Serverbezogene Profile
Schalte Konfigurationen automatisch um, wenn du unterschiedliche Server betrittst:

```javascript
on('serverJoin', (event) => {
  if (event.ip.includes('pvp.example.com')) {
    loadConfig('server-bypass');
  } else {
    loadConfig('ghost-default');
  }
});
```
