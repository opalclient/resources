<KeyTakeaways>

- Fabricインストーラは、公式のFabricインストーラという一つのツールでインストールされます。
- これにより、新しい起動プロファイルが追加され、バニラ版のゲームは一切変更されません。
- `mods`フォルダは、Fabricプロファイルを一度起動すると表示されます。
- プレイするMinecraftのバージョンと、インストーラを完全に一致させてください.

</KeyTakeaways>

[Fabric](https://fabricmc.net/) ローダーのインストールには、ツールが一つと、いくつかのクリックだけで済みます。公式のFabricインストーラーをダウンロードし、実行して、使用したいMinecraftのバージョンを選択し、新しい起動プロファイルを作成させてください。ランチャーを開き、そのFabricプロファイルを選択して、ゲームを一度起動します。ローダーはこれで有効になり、Modに対応する準備が整いました.

<Callout variant="info" title="どこ Opal 合う">OpalはFabricのModなので、ここでインストールする同じローダーに依存しています。ローダーが設置されたら、[セットアップガイド](/ja/docs/setup)を参照して、正確なバージョンと推奨されるJVM引数を設定してください</Callout>

## 手順

<Steps>

<Step number={1} title="インストーラーをダウンロードしてください">Fabric プロジェクトのサイトから、公式の [Fabric](https://fabricmc.net/) インストーラーを入手してください。それは小さな Java プログラムです</Step>

<Step number={2} title="走らせろ">ジャーをダブルクリックするか、ダブルクリックしても何も起こらない場合は、ターミナルから実行してください</Step>

<Step number={3} title="お好きなものをどうぞ Minecraft バージョン">プレイする正確なバージョンを選んでください。最近発売されたものや、まだ使っている古いバージョンなどです</Step>

<Step number={4} title="インストール">インストーラーは、あなたのランチャーに新しいプロファイルを作成し、対応するローダーファイルをダウンロードします</Step>

<Step number={5} title="選択してください Fabric プロフィール">Minecraftランチャーを開き、インストーラーが作成したプロファイルに切り替えてください</Step>

<Step number={6} title="一度起動">そのプロファイルでゲームを開始してください。ローダーがファイルをセットアップし、`mods` フォルダを作成します</Step>

</Steps>

## インストーラが実際に何をするか

インストーラはゲームのjarファイルをパッチ適用しません。代わりに、Minecraftが起動する前に[Fabric](https://fabricmc.net/)ローダーを起動する起動プロファイルを追加します。そのローダーは起動時に`mods`フォルダを読み込み、各MODをロードします。プロファイルがバニラ版の隣に配置されているため、MODを適用しないゲームはそのままの状態を保たれ、いつでも元の状態に戻すことができます。

## インストーラとランチャーのサポート

ローダーを導入する方法は2つあります。自分の環境に合った方を選んでください。

| 方法 | 使うべき時 | 備考 |
|--------|----------------|-------|
| 公式Fabricインストーラ | 標準ランチャーのデフォルト | プロファイルを書き込み、ローダーをダウンロードします。数分で完了します |
| Fabric内蔵ランチャー | すでにサードパーティ製のランチャーを使用している場合 | ランチャーがバージョンマッチングを処理します |

どちらの方法でも、最終的にはFabricプロファイルを起動前に選択することになります。

## ファイルが配置される場所

| OS | パス |
| :--- | :--- |
| Windows | `%appdata%\.minecraft` |
| macOS | `~/Library/Application Support/minecraft` |
| Linux | `~/.minecraft` |

`mods`フォルダは、Fabricを初めて起動した後にそのディレクトリ内に表示されます。フォルダが存在したら、そこにMODのjarファイルをドロップしてください。

## ローダーがインストールされない場合

<Checklist>
<ChecklistItem>インストーラーが起動しません。Javaが不足しているか、jarファイルが誤ったプログラムで開かれている可能性があります。最新のJavaランタイムをインストールして、再度お試しください.</ChecklistItem>
<ChecklistItem>ランチャーにFabricプロファイルが見当たらない場合：ランチャーを更新または再起動し、バージョン選択肢から新しいプロファイルを探してください.</ChecklistItem>
<ChecklistItem>MODを追加するとゲームがクラッシュします。ローダーは問題ありませんが、MODが別のMinecraftのバージョンを対象としています。インストールしたバージョンにMODをすべて合わせましょう.</ChecklistItem>
<ChecklistItem>Minecraftのバージョンが間違っています。インストーラーを再度実行し、正しいバージョンを選択してください。複数のプロファイルを作成できます.</ChecklistItem>
</Checklist>

## よくある質問

<FAQList>

<FAQItem question="Forgeの導入はFabricのインストールに必要ですか？">
いいえ。Fabricは独立したModローダーです。プロファイルごとにどちらか一方をインストールしてください。両方を同時にインストールする必要はありません。
</FAQItem>

<FAQItem question="ローダーのインストールはバニラ版Minecraftに変更を加えますか？">
いいえ。インストーラーは新しいプロファイルを追加するだけで、バニラ版のゲームはそのままです。プロファイルを切り替えることで、元の状態に戻せます。
</FAQItem>

<FAQItem question="ローダーをインストールした後、MODはどこに移動しますか？">
Minecraftのディレクトリ内の`mods`フォルダに作成されます。Fabricプロファイルを初めて起動したときに作成されます。
</FAQItem>

<FAQItem question="公式インストーラーなしでローダーをインストールできますか？">
はい。いくつかのランチャーはFabricのサポートを追加し、スタンドアロンインストーラーを完全にスキップします。
</FAQItem>

</FAQList>
