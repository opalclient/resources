<KeyTakeaways>

- Fabric mod の導入は、ローダー、Fabric API、そして mod jar の 3 段階です。
- ローダー、API、mod のいずれも、同じ Minecraft のバージョンを対象とする必要があります。
- mod は、Minecraft ディレクトリ内の mods フォルダに置きます。
- 多くの導入失敗の原因は、バージョンの不一致、または Fabric API の不足です.

</KeyTakeaways>

ファブリック（[Fabric](https://fabricmc.net/)）のModをインストールするには、3つのステップが必要です。まず、Minecraftの正確なバージョンに対応したFabricローダーをインストールし、次にFabric APIをmodsフォルダに配置します。最後に、Modのjarファイルをその隣に置きます。Fabricプロファイルを起動すると、Modがロードされます。インストールで問題が発生する主な原因はバージョン不一致なので、すべてのファイルをMinecraftの同じバージョンに合わせるようにしてください.

Opal は以下の手順に従います。これは Fabric の Mod です。正確なバージョンと推奨される JVM 引数の設定については、[セットアップガイド](/ja/docs/setup) を参照してください。
</Callout>

## 三つのステップ

<Steps>

<Step number={1} title="Fabricローダーのインストール">
Minecraftのバージョンに合わせて、Fabricローダーを公式のFabricインストーラーまたはFabricに対応したランチャーを使用してインストールしてください。詳細な手順は[Fabricローダーのインストール方法](/ja/learn/how-to-install-fabric-loader)をご覧ください。
</Step>

<Step number={2} title="FabricAPIを追加する">[Fabric API](/ja/learn/what-is-the-fabric-api) のJARファイルを、modsフォルダに置いてください。ほとんどのMODはその依存関係となっています</Step>

<Step number={3} title="モッドを追加する">モッズのJARファイルを、Fabric APIの隣にある同じモッズフォルダに置いてください</Step>

</Steps>

Fabric プロファイルを起動し、mod は起動時にロードされます。表示されない場合は、以下のトラブルシューティングリストを確認してください。

## mod フォルダの場所

| OS | パス |
| :--- | :--- |
| Windows | `%appdata%\.minecraft\mods` |
| macOS | `~/Library/Application Support/minecraft/mods` |
| Linux | `~/.minecraft/mods` |

まだ `mods` フォルダが存在しない場合は、Fabric プロファイルを一度実行すると、自動的に作成されます。

## mod がロードされない場合

<Checklist>
<ChecklistItem>Modが表示されません：JARファイルが間違ったフォルダにあるか、Fabric以外のプロファイルで起動しました.</ChecklistItem>
<ChecklistItem>起動時にクラッシュし、クラスが見つからないエラーが発生しました。Fabric API が存在しないか、バージョンが正しくありません.</ChecklistItem>
<ChecklistItem>バージョン不一致：ローダー、Fabric API、およびmodはすべて同じMinecraftバージョンを対象とする必要があります.</ChecklistItem>
</Checklist>

## よくある質問

<FAQList>

<FAQItem question="すべてのMODにFabric APIは必要ですか？">
ほとんどのFabric MODはそれに依存しているため、デフォルトでインストールしてください。ごく一部のスタンドアロンMODは不要ですが、追加しても害はありません。
</FAQItem>

<FAQItem question="起動時にMODがクラッシュする原因は何ですか？">
通常はバージョン不一致、またはFabric APIが不足していることが原因です。ローダー、Fabric API、そしてMODがすべて同じMinecraftバージョンを対象としているか確認してください。
</FAQItem>

<FAQItem question="ファブリックMODはどこに置けばいいですか？">
Minecraftのディレクトリ内の`mods`フォルダに置いてください（上記のパスをご参照ください）。フォルダが存在しない場合は、Fabricプロファイルを一度実行してください。
</FAQItem>

</FAQList>
