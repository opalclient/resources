良いFabric環境を構築するための基本的なmodはいくつかあります。Fabric APIをベースとし、Sodiumでパフォーマンスを向上させ、Irisでシェーダーを使用し、ModMenuで全てを管理します。これらのmodを最初にインストールすれば、ほとんどの他のmodも問題なく動作し、クライアントも同様です。Opalユーザーの方はご注意ください。Lithiumはよく選ばれますが、Opalとの互換性がないため、インストールしない方が良いでしょう.

<KeyTakeaways>

- Fabric API がベースです。他のものを取り付ける前に、これをインストールしてください。
- FPS には Sodium、シェーダーには Iris、ModMenu で全てを管理しましょう。
- Lithium は単独で使用しても良いですが、Opal とは互換性がありません。
- Opal はクライアント層です。他のものと同じフォルダにインストールします.

</KeyTakeaways>

## Fabric API

ほとんどのFabric製MODが依存する共有ライブラリです。まずインストールしてください。これがないと、ほとんどのMODがロードされません。機能というよりは、必須と言えるでしょう。

## Sodium

FPSを向上させるレンダリングの書き換えです。特に弱いGPUに負荷がかかります。導入することで最大のパフォーマンス向上が得られ、Opalと併用しても問題ありません。

## Iris

シェーダーパックを実行し、Sodiumとの連携を前提に構築されているため、フレームレートを犠牲にすることなく美しいグラフィックを楽しめます。シェーダーに興味がない場合はスキップしても構いません。

## ModMenu

ゲーム内のMODリストと設定画面を追加し、ファイルを編集することなくインストールされているMODを確認したり、設定を変更したりできます。他のMODとの相性が格段に良くなる、生活の質を向上させるツールです。

## Lithiumについて

[Lithium](https://modrinth.com/mod/lithium)はゲームロジックを最適化し、バニラプラス環境では非常に効果的です。ただし、Opalとの互換性がないため、Opalがフォルダにある場合はLithiumを導入せず、Sodiumにパフォーマンスを頼ってください。

## 最上層

スターターパックが導入されれば、フォルダはクライアントの準備が整います。OpalはMinecraft 1.21.10のFabric Modなので、他のModと同様に`mods`フォルダにインストールします。パフォーマンスとシェーダースタックを維持し、200以上の機能、ガラスUI、そしてスクリプティングエンジンを追加します。
