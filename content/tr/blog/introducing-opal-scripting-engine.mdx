Opal, GraalVM JavaScript motorunu, Minecraft istemcisinin içinde çalışan tam bir JS çalışma zamanını gönderiyor. Oyun açıkken Opal'ın dahili API'leriyle etkileşim kuran komut dosyaları yazarsınız. Bu bir makro kaydedici veya bir kısayol sistemi değildir. İstemcinin olay sistemine, modül ayarlarına ve oyun durumuna erişebilen gerçek bir programlama ortamıdır.

<KeyTakeaways>

- Tam bir GraalVM JavaScript çalışma zamanı, yalnızca bir makro kaydedici değil.
- Oyun olaylarına tepki verin, modül ayarlarını değiştirin ve kendi HUD'unuzu çizin.
- Betikler çalışma zamanında yüklenir, yeniden başlatmaya gerek yoktur ve Opal'ın API'lerine göre güvenli bir şekilde çalışır.
- Diğer çoğu istemci temel makrolarda durur.

</KeyTakeaways>

## Neden GraalVM

GraalVM, Oracle tarafından sunulan çok dilli bir çalışma zamanıdır. Bunu seçmemizin somut nedenleri var:

- Hız: JIT derlemesi, betikleri her döngüde yeterince hızlı çalışır hale getirir.
- JavaScript: Çoğu kişi zaten biliyor, bu nedenle öğrenme eğrisi kısadır.
- İş parçacığı güvenliği: JS-Java bağlamaları, sizi takılıp düşmekten alıkoymak yerine eşzamanlılığı düzgün bir şekilde yönetir.
- Tanıdık kalıplar: Standart JavaScript API'leri beklendiği gibi davranır.

## Neler Yapabilirsiniz

### Özel otomasyon

Oyun içi olaylara yanıt verin:

```javascript
// Auto-respond to certain chat messages
on('chat', (event) => {
  if (event.message.includes('1v1?')) {
    sendChat('/duel accept');
  }
});
```

### Dinamik modül yapılandırması

Oyun durumuna göre modül ayarlarını değiştirin:

```javascript
// Increase reach when health is low
on('tick', () => {
  const health = player.getHealth();
  if (health < 8) {
    modules.get('Reach').setSetting('distance', 3.3);
  } else {
    modules.get('Reach').setSetting('distance', 3.1);
  }
});
```

### Özel HUD öğeleri

Kendi üst üste bindirmelerinizi çizin:

```javascript
// Display a combo counter on screen
let combo = 0;
on('attackEntity', () => { combo++; });
on('damage', () => { combo = 0; });
on('render2d', (ctx) => {
  ctx.drawText(`Combo: ${combo}`, 10, 10, 0xFFFFFF);
});
```

### Sunucuya özel profiller

Farklı sunuculara katıldığınızda yapılandırmaları otomatik olarak değiştirin:

```javascript
on('serverJoin', (event) => {
  if (event.ip.includes('pvp.example.com')) {
    loadConfig('server-bypass');
  } else {
    loadConfig('ghost-default');
  }
});
```
