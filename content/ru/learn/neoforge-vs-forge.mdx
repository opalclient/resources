## Что из чего

Forge — это оригинальный загрузчик модов для Minecraft. Он поддерживал большую часть сцены модопаков начиная с версии 1.6, с использованием аннотаций для обработки событий, определенного жизненного цикла и ASM-трансформаторов классов в основе.

NeoForge — это форк Forge, отделившийся в 2023 году, когда конфликт руководства в проекте Forge достиг кульминации. Он сохраняет ту же общую архитектуру (реестры, возможности, обработчик событий), но управляется сообществом с более открытой моделью внесения вклада.

## Почему произошел форк

Разделение было связано с управлением, а не с переписыванием. Группа разработчиков Forge перешла на проект, принадлежащий сообществу, после разногласий по поводу того, как управляется Forge. Поскольку NeoForge был создан на основе кодовой базы Forge, два загрузчика разделяют большинство своих основных идей, и авторы модов, знакомые с Forge, сразу узнают NeoForge.

Со временем они стали расходиться. NeoForge переработал части API и внедрил собственные соглашения, поэтому мод, созданный для текущей версии NeoForge, не гарантированно загрузится на Forge, и наоборот.

## Чем они отличаются

| | NeoForge | Forge |
| :--- | :--- | :--- |
| Происхождение | Форк Forge, созданный сообществом в 2023 году | Оригинальный загрузчик |
| Управление | Принадлежит сообществу, открытые вклады | Традиционное руководство проектом Forge |
| Архитектура | Общая база (реестры, возможности, шина событий) | Реестры, возможности, шина событий |
| Ориентация на новые версии | Текущие версии Minecraft | Поддерживает старую моддинговую сцену |
| Каталог модов | Активно расширяется для современных версий | Обширный каталог для старых версий |

## Какой выбрать

<Checklist>
<ChecklistItem>Выберите NeoForge, если вы играете в актуальные версии Minecraft и ваши моды перешли на него. Большинство активно поддерживаемых Forge-модов теперь ориентированы на NeoForge в новых релизах.</ChecklistItem>
<ChecklistItem>Выберите Forge, если вы используете старый модпак или контент-мод, который распространяется только для Forge. Многие проверенные временем сборки до сих пор построены на его основе.</ChecklistItem>
<ChecklistItem>Проверьте страницу версий для каждого мода. Один и тот же автор может выпускать Forge-версию для старых версий и NeoForge-версию для новых.</ChecklistItem>
</Checklist>

## Часто задаваемые вопросы

<FAQList>

<FAQItem question="Является ли NeoForge заменой Forge?">
В текущих версиях Minecraft большая часть сообщества рассматривает NeoForge как преемника модов Forge-линейки. Forge всё ещё существует и продолжает поддерживать более старые версии, поэтому ни один из них полностью не вытеснил другой.
</FAQItem>

<FAQItem question="Можно ли использовать моды Forge в NeoForge?">
Не всегда надежно. Они имеют общие корни, поэтому некоторые простые моды могут работать, но API разошлись. Мод, предназначенный для одного загрузчика, не гарантированно загрузится в другом. Установите сборку, соответствующую вашему загрузчику.
</FAQItem>

<FAQItem question="Являются ли NeoForge и Fabric одним и тем же?">
Нет. NeoForge происходит из семейства Forge и разделяет его модель шины событий и реестра. [Fabric](https://fabricmc.net/) — это отдельный, более лёгкий загрузчик со своей собственной моделью патчинга на основе миксинов. Это разные экосистемы.
</FAQItem>

</FAQList>
