<KeyTakeaways>

- Установка Fabric mod состоит из трех шагов: загрузчик, Fabric API и затем jar-файл мода.
- Каждый компонент (загрузчик, API, мод) должен быть предназначен для одной и той же версии Minecraft.
- Моды располагаются в папке "mods" внутри вашей директории Minecraft.
- Большинство неудачных установок связаны с несовпадением версий или отсутствием Fabric API.

</KeyTakeaways>

Установка мода для [Fabric](https://fabricmc.net/) состоит из трех шагов: установите Fabric Loader для вашей точной версии Minecraft, поместите Fabric API в папку с модами, а затем поместите jar-файл мода рядом с ней. Запустите профиль Fabric, и мод загрузится. Большинство проблем с установкой возникают из-за несовпадения версий, поэтому убедитесь, что все компоненты соответствуют одной и той же версии Minecraft.

## Три шага

<Steps>

<Step number={1} title="Установите Fabric loader">
Установите загрузчик [Fabric](https://fabricmc.net/) для вашей версии Minecraft, используя официальный установщик Fabric или лаунчер, поддерживающий Fabric. Подробная инструкция по установке загрузчика Fabric доступна [здесь](/ru/learn/how-to-install-fabric-loader).
</Step>

<Step number={2} title="Добавьте Fabric API">Поместите файл [Fabric API](/ru/learn/what-is-the-fabric-api) в папку с модами. Большинство модов от него зависят</Step>

<Step number={3} title="Добавь мод">Поместите jar-файл мода в ту же папку с модами, рядом с Fabric API</Step>

</Steps>

Затем запустите профиль Fabric, и мод загрузится при запуске. Если он не появляется, проверьте список решений проблем ниже.

## Где находится папка с модами

| ОС | Путь |
| :--- | :--- |
| Windows | `%appdata%\.minecraft\mods` |
| macOS | `~/Library/Application Support/minecraft/mods` |
| Linux | `~/.minecraft/mods` |

Если папка `mods` еще не существует, запустите профиль Fabric один раз, и она будет создана для вас.

## Когда мод не загружается

<Checklist>
<ChecklistItem>Мод не отображается: файл мода находится не в той папке, или вы запустили профиль, не использующий Fabric.</ChecklistItem>
<ChecklistItem>Игра вылетает при запуске с ошибкой отсутствующего класса: отсутствует Fabric API или установлена неверная версия.</ChecklistItem>
<ChecklistItem>Несоответствие версий: загрузчик, Fabric API и мод должны быть нацелены на одну и ту же версию Minecraft.</ChecklistItem>
</Checklist>

## Часто задаваемые вопросы

<FAQList>

<FAQItem question="Нужен ли Fabric API для каждого мода?">
Большинство модов для Fabric зависят от него, поэтому установите его по умолчанию. Небольшое количество автономных модов не нуждаются в нём, но добавление его никогда не повредит.
</FAQItem>

<FAQItem question="Почему моя модификация вылетает при запуске?">
Обычно это связано с несовместимостью версий или отсутствием Fabric API. Убедитесь, что загрузчик, Fabric API и сама модификация предназначены для одной и той же версии Minecraft.
</FAQItem>

<FAQItem question="Куда класть моды для Fabric?">
В папку `mods`, расположенную внутри вашей директории Minecraft (пути указаны выше). Запустите профиль Fabric один раз, если папки еще нет.
</FAQItem>

</FAQList>
