<KeyTakeaways>

- Fabric Loader устанавливается с помощью одного инструмента: официального установщика Fabric.
- Он добавляет новый профиль запуска и никогда не изменяет вашу оригинальную игру.
- Папка `mods` появляется после первого запуска профиля Fabric.
- Убедитесь, что Fabric Loader соответствует точной версии Minecraft, в которую вы планируете играть.

</KeyTakeaways>

Установка загрузчика [Fabric](https://fabricmc.net/) требует всего одного инструмента и нескольких кликов. Скачайте официальный установщик Fabric, запустите его, выберите нужную версию Minecraft и позвольте ему создать новый профиль запуска. Откройте ваш лаунчер, выберите этот профиль Fabric и запустите игру один раз. Загрузчик теперь активен и готов к модам.

## Что на самом деле делает установщик

Установщик не патчит игровой jar-файл. Он добавляет профиль запуска, который загружает Fabric-загрузчик перед запуском Minecraft. Этот загрузчик при запуске читает вашу папку `mods` и загружает каждый мод. Поскольку профиль находится рядом с оригинальным, ваша немодифицированная игра остаётся нетронутой, и вы можете вернуться к ней в любое время.

## Установщик против поддержки лаунчера

У вас есть два способа установить загрузчик. Выберите тот, который подходит для вашей конфигурации.

| Метод | Когда использовать | Примечания |
|--------|----------------|-------|
| Официальный установщик Fabric | По умолчанию для стандартного лаунчера | Создаёт профиль, скачивает загрузчик, делается за несколько минут |
| Лаунчер со встроенной поддержкой Fabric | Вы уже используете сторонний лаунчер | Лаунчер сам подбирает версию |

Оба варианта заканчиваются одинаково: профилем Fabric, который вы выбираете перед запуском.

## Где находятся файлы

| ОС | Путь |
| :--- | :--- |
| Windows | `%appdata%\.minecraft` |
| macOS | `~/Library/Application Support/minecraft` |
| Linux | `~/.minecraft` |

Папка `mods` появляется в этой директории после первого запуска Fabric. Поместите туда jar-файлы модов, как только она появится.

</Checklist>
