<KeyTakeaways>

- ClickGUI — это меню, управляемое с помощью щелчков мыши, которое использует клиент утилиты для Minecraft для переключения модулей и редактирования их настроек.
- Оно открывается по нажатию комбинации клавиш и отображает перетаскиваемые панели, сгруппированные по категориям, содержащие переключатели, слайдеры и выпадающие списки, а также сохраняет раскладку в файл конфигурации.
- Это меню, а не особенность игрового процесса, поэтому само по себе не несёт риска обнаружения.

</KeyTakeaways>

ClickGUI — это внутриигровое меню, которое использует клиент утилиты для Minecraft, чтобы позволить вам включать и выключать модули и редактировать их настройки с помощью мыши, а не вводить чат-команды по одной. Оно открывается по специальному сочетанию клавиш и представляет набор функций клиента в виде перетаскиваемых окон, сгруппированных по категориям: флажки для переключателей, слайдеры для чисел, выпадающие списки для режимов, подбор цвета для всего визуального. Для многих игроков это единственный интерфейс, с которым они взаимодействуют, поскольку он показывает всё, что может делать клиент, вместо того, чтобы требовать запоминать названия команд перед их использованием.

## Как это работает

ClickGUI построен как набор панелей, обычно по одной на категорию, которые можно перетаскивать по экрану и сворачивать или разворачивать по желанию. Каждая панель перечисляет свои модули в виде строк: переключатель для включения или выключения модуля и стрелка для разворачивания этой строки, чтобы отобразить его фактические настройки. Разверните одну, и вы получите реальные элементы управления: флажок для простого значения включения/выключения, слайдер для числа, например, диапазона или скорости, выпадающий список или циклический элемент управления для режима, подбор цвета для всего визуального. Ничего из этого не для красоты. Эти элементы управления считывают и записывают текущие настройки модуля, поэтому перетаскивание слайдера меняет поведение в тот момент, когда вы отпускаете мышь. Не нужно повторно открывать меню, не нужно повторно вводить команду.

Макет не удаляется при закрытии меню. Большинство клиентов записывают позиции панелей и состояние открытия или закрытия прямо в файл конфигурации, поэтому графический интерфейс возвращается к тому виду, в котором вы его оставили, от сессии к сессии. Создайте макет, который вам нравится, один раз, и он останется с вами.

## ClickGUI против Tab GUI

Некоторые клиенты сочетают ClickGUI со вторым меню, работающим только с клавиатурой, обычно называемым Tab GUI, и это намеренное разделение, а не избыточное.

| | ClickGUI | Tab GUI |
|---|---|---|
| Ввод | Мышь | Клавиатура |
| Взаимодействие | Перетаскивание панелей, щелчки переключателей, перемещение ползунков | Стрелки или циклический выбор клавишей Tab по плоскому списку |
| Лучше всего подходит для | Просмотра полного списка модулей, настройки значений на ощупь | Переключение модуля, не отрывая руки от WASD |

Один не заменяет другой. ClickGUI выигрывает в открытии: просмотре категорий, чтении того, что на самом деле делает ползунок, настройке значения визуально с поисковой строкой перед вами. Tab GUI выигрывает в разгаре боя, когда ваши руки уже на клавиатуре, и достижение мыши стоит вам секунды, которую вы не можете себе позволить. Клиенты, которые поставляют оба, делают ставку на то, что вам понадобится каждый из них в разные моменты, и обычно они правы.
