<KeyTakeaways>

- Fabric API — это общая библиотека хуков и вспомогательных инструментов, на основе которой создаются большинство модов для Fabric.
- Она намеренно отделена от загрузчика: небольшой загрузчик обновляется быстро, а API развивается независимо.
- Она предоставляет события, сетевые возможности, реестр и вспомогательные инструменты рендеринга, которыми пользуются многие моды.
- Отсутствие или устаревшая версия Fabric API — наиболее частая причина сбоя мода при запуске.

</KeyTakeaways>

Fabric API — это общая библиотека хуков и вспомогательных инструментов, на основе которой создаются большинство модов для [Fabric](https://fabricmc.net/). Основной загрузчик Fabric намеренно небольшой, поэтому API заполняет пробелы: события, сетевое взаимодействие, утилиты реестра, инструменты рендеринга. Если мод Fabric аварийно завершает работу при запуске с ошибкой отсутствующего класса, причиной, как правило, является отсутствующая или устаревшая Fabric API.

</Callout>

## Почему оно отделено от загрузчика

Fabric намеренно разделяет систему на две части. Загрузчик остается небольшим и стабильным, а Fabric API поставляется с большим, более динамично развивающимся набором общих интерфейсов. Такое разделение позволяет загрузчику быстро обновляться до новой версии Minecraft, в то время как API развивается в своем собственном темпе.

## Что оно предоставляет

API предоставляет модам общий, версионно-стабильный способ делать то, что требуется многим модам:

- События жизненного цикла и игрового процесса, к которым можно подключиться.
- Сетки-помощники для пользовательских пакетов.
- Помощники реестра и контента для добавления предметов, блоков и сущностей.
- Помощники рендеринга и ресурсов, которыми делятся моды.

Без этого каждый мод бы изобретал эти компоненты заново и ломал друг друга.

## Нужно ли оно вам?

Почти всегда, да. Большинство модов Fabric указывают Fabric API в качестве зависимости, поэтому вы устанавливаете его один раз как мод наряду с другими. Убедитесь, что версия API соответствует вашей версии Minecraft.
