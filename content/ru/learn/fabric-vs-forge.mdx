<KeyTakeaways>

- Fabric — это легкий, быстро обновляемый загрузчик, предпочитаемый модами, ориентированными на производительность и функциональность.
- Forge — это более старый, более тяжелый загрузчик с самым обширным каталогом крупных модов.
- Выбирайте Fabric для поддержки ранних версий и небольшого размера; выбирайте Forge для больших модпаков.
- Мод нацелен на один из загрузчиков, поэтому подбирайте моды под тот загрузчик, который вы используете.

</KeyTakeaways>

Fabric — это лёгкий и быстрый загрузчик модов, который поддерживает новые версии Minecraft в течение нескольких дней и пользуется популярностью у модов, улучшающих производительность и функциональность. Forge — это более старый и громоздкий загрузчик с самым обширным каталогом крупных контентных модов. Выбирайте Fabric для скорости и поддержки новых версий; выбирайте Forge для больших модпаков и контентных модов, которые распространяются только там.

## Что из себя представляет каждый из них

Fabric — это модульный загрузчик модов, построенный вокруг небольшого ядра и отдельно версионируемого [Fabric API](/ru/learn/what-is-the-fabric-api). Он известен тем, что быстро обновляется до новых версий Minecraft и использует чистую модель патчинга на основе Mixins.

Forge — это хорошо зарекомендовавший себя загрузчик, который в течение многих лет поддерживал большинство крупных модопаков. Он поставляется со своей собственной широкой API, что упрощает создание модов с большим объемом контента, но замедляет обновление загрузчика до каждой версии.

## В чем разница

| | Fabric | Forge |
| :--- | :--- | :--- |
| Вес | Легкое ядро + Fabric API | Тяжелый, большой встроенный API |
| Поддержка новых версий | Дни после релиза | Недели или месяцы |
| Силен в | Производительности + утилитах | Большом контенте + модопаках |
| Модель патчинга | Mixins | Coremods и собственные хуки |
| Совместимость между модами | API для каждого мода | Широкий общий API |

## Какой выбрать

- Выберите Fabric, если вам нужна самая новая версия Minecraft как можно раньше, вы хотите запускать моды для повышения производительности или вам нужен минимальный объем ресурсов.
- Выберите Forge, если ваши любимые контент-моды или модопаки работают только на Forge, или вам нужен один большой общий API для разработки.
- Многие игроки используют оба, создавая отдельные профили для каждого загрузчика, поскольку большинство модов ориентированы на один или другой.

## FAQ

<FAQList>

<FAQItem question="Работает ли Fabric быстрее, чем Forge?">
Сам загрузчик легче, а популярные модификации, повышающие производительность, которые с ним работают (Sodium, Lithium), работают на Fabric. Сырая скорость зависит от установленных вами модов, а не только от загрузчика.
</FAQItem>

<FAQItem question="Можно ли использовать моды Forge на Fabric?">
Не напрямую. Мод, созданный для одного загрузчика, не работает с другим без слоя совместимости, поэтому перед установкой убедитесь, какой загрузчик использует мод.
</FAQItem>

<FAQItem question="Какие обновления выходят быстрее?">
Fabric обычно готов через несколько дней после выхода новой версии Minecraft. Forge обычно занимает больше времени из-за его большей площади API.
</FAQItem>

</FAQList>
