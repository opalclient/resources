<KeyTakeaways>

- Quilt — это форк сообщества 2021 года, основанный на [Fabric](https://fabricmc.net/), который запускает Fabric-моды без изменений.
- Fabric — это оригинальный загрузчик с гораздо более обширной экосистемой модов и пользовательской базой.
- Совместимость односторонняя: Quilt загружает Fabric-моды, но Fabric не может загружать моды, предназначенные только для Quilt.
- Для большинства игроков Fabric — это безопасный выбор по умолчанию; Quilt подходит тем, кто хочет воспользоваться его дополнительными API.

</KeyTakeaways>

Quilt — это форк сообщества, основанный на [Fabric](https://fabricmc.net/), который поддерживает двоичную совместимость с модами для Fabric, добавляя при этом собственные библиотеки. Fabric — это оригинальный загрузчик, обладающий гораздо более обширной экосистемой модов и пользовательской базой. Для большинства игроков Fabric — это безопасный выбор по умолчанию. Quilt подходит тем, кто хочет использовать его дополнительные API или поддерживает форк по принципу.

## Что из чего

Fabric — это легковесный, модульный загрузчик модов, поддерживаемый FabricMC. Он сочетает в себе небольшой основной модуль с отдельно версионируемым [Fabric API](/ru/learn/what-is-the-fabric-api) и использует модель патчинга на основе миксинов. Он быстро обновляется до новых версий Minecraft и содержит самый большой каталог модов, повышающих производительность и функциональность.

Quilt — это форк Fabric, созданный в 2021 году после разногласий по вопросам управления внутри FabricMC. Он запускает моды Fabric без изменений, поставляется со своим собственным Quilted Fabric API в качестве слоя совместимости с Fabric API и добавляет QSL, Quilt Standard Libraries, для своих собственных API. Его распространение невелико по сравнению с Fabric.

## В чем разница

Основной практический момент заключается в направлении совместимости. Quilt может загружать моды Fabric, но Fabric не может загружать моды, зависящие от функций QSL, специфичных для Quilt. Таким образом, мод, написанный для обычного Fabric, достигает обеих аудиторий, в то время как мод, зависящий от QSL, достигает только пользователей Quilt.

| | Fabric | Quilt |
| :--- | :--- | :--- |
| Происхождение | Оригинальный загрузчик от FabricMC | Форк Fabric сообществом в 2021 году |
| Экосистема модов | Самая большая в современных версиях Minecraft | Запускает моды Fabric и моды QSL |
| Собственный API | Fabric API | Quilted Fabric API (совместимость) + QSL |
| Совместимость | Не может загружать моды, специфичные для Quilt | Загружает моды Fabric без изменений |
| Распространение | Очень большое | Небольшое по сравнению с Fabric |
| Модель патчинга | Миксины | Миксины (унаследованы от Fabric) |

## Какой выбрать

<Checklist>
<ChecklistItem>Выберите Fabric, если вам нужен самый широкий выбор модов, самая быстрая поддержка обновлений и загрузчик, на который по умолчанию ориентируются большинство клиентов и модов.</ChecklistItem>
<ChecklistItem>Выберите Quilt, если вам нужны его функции QSL, запустить мод, предназначенный только для Quilt, или если вы предпочитаете управление форком. Ваши существующие моды Fabric всё ещё работают.</ChecklistItem>
<ChecklistItem>Вы не запускаете оба одновременно. Выберите один загрузчик для каждого профиля, и поскольку Quilt читает моды Fabric, переключение не потребует больших усилий в любом случае.</ChecklistItem>
</Checklist>

## Часто задаваемые вопросы

<FAQList>

<FAQItem question="Совместим ли Quilt с модами Fabric?">
Да. Quilt сохраняет двоичную совместимость и загружает моды Fabric через свой слой Quilted Fabric API. Обратное неверно: Fabric не может запускать моды, требующие QSL API, специфичные для Quilt.
</FAQItem>

<FAQItem question="Почему существует Quilt, если Fabric уже работает?">
Quilt отделился от FabricMC в 2021 году из-за разногласий по вопросам управления. Он стремится оставаться совместимым с Fabric, предлагая при этом собственный дизайн API через QSL.
</FAQItem>

<FAQItem question="Стоит ли клиенту утилиты ориентироваться на Fabric или Quilt?">
В большинстве случаев стоит ориентироваться на Fabric, поскольку у него больше пользовательская база и поддержка версий, а сборка Fabric также работает на Quilt. Создание сборки только для Quilt в большинстве случаев уменьшит аудиторию без какой-либо выгоды.
</FAQItem>

<FAQItem question="Один быстрее другого?">
Различия незначительные. Quilt наследует модель миксинов Fabric, поэтому производительность зависит от установленных модов, а не от загрузчика.
</FAQItem>

</FAQList>
