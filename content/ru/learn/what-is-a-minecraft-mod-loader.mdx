<KeyTakeaways>

- Мод-лоадер — это слой, который позволяет Minecraft запускать моды; в ванильной версии поддержки модов нет.
- Он патчит игру при запуске и предоставляет каждому моду возможность подключиться к коду.
- Вы устанавливаете его один раз для вашей версии Minecraft, а затем помещаете файлы модов (jar) в папку mods.
- Основные варианты: Fabric, Forge, NeoForge и Quilt; выбирайте тот, который поддерживается вашими модами.

</KeyTakeaways>

Мод-лоадер — это слой, который позволяет Minecraft запускать моды. Ванильный Minecraft не поддерживает моды, поэтому лоадер патчит игру при запуске и предоставляет модам возможность к ней подключаться. Вы устанавливаете лоадер один раз для вашей версии Minecraft, затем помещаете файлы модов (jar) в папку "mods" и они загружаются при запуске.

Opal — это мод для [Fabric](https://fabricmc.net/), поэтому Fabric — это загрузчик, который вам нужен. [Руководство по настройке](/ru/docs/setup) описывает этот процесс.
</Callout>

## Почему вам нужен загрузчик

Моды изменяют поведение игры, а значит, должны работать внутри игрового кода. Загрузчик делает это возможным. Без него мод jar — это просто файл, который игра игнорирует.

## Как это работает

При запуске загрузчик вставляется перед началом игры, читает папку с модами и подключает каждый мод к нужным точкам. [Fabric](https://fabricmc.net/) делает это с помощью mixins — подхода, основанного на целевом патчинге байт-кода. Forge использует собственную, более крупную систему точек подключения. В любом случае, игра запускается с вшитыми модами.

## Основные загрузчики

| Загрузчик | Известен |
| :--- | :--- |
| Fabric | Легковесный, быстро обновляется, моды для производительности и утилиты |
| Forge | Моды с большим количеством контента и давно существующие модпаки |
| NeoForge | Форк Forge, управляемый сообществом |
| Quilt | Форк Fabric с дополнительными функциями |

## Выбор загрузчика

Используйте загрузчик, для которого предназначены ваши моды. Большинство модов распространяются для одного загрузчика, а не для обоих, поэтому моды, которые вы хотите использовать, обычно сами решают, какой загрузчик использовать. Для более глубокого анализа см. [Fabric vs Forge](/ru/learn/fabric-vs-forge).
