Opal поставляется с движком JavaScript GraalVM, полноценной средой выполнения JS, работающей внутри клиента Minecraft. Вы пишете скрипты, которые взаимодействуют с внутренними API Opal во время игры. Это не макро-рекордер и не система привязки клавиш. Это настоящая среда программирования с доступом к системе событий клиента, конфигурациям модулей и состоянию игры.

<KeyTakeaways>

- Полная среда выполнения JavaScript GraalVM, а не просто макро-рекордер.
- Реагируйте на игровые события, изменяйте настройки модулей и рисуйте собственный HUD.
- Скрипты загружаются во время выполнения, без перезапуска, и работают в изолированной среде с использованием API Opal.
- Большинство других клиентов ограничиваются базовыми макросами.

</KeyTakeaways>

## Почему GraalVM
GraalVM — это полиглотный runtime от Oracle. Мы выбрали его по конкретным причинам:

- Скорость: JIT-компиляция поддерживает скрипты достаточно быстрыми, чтобы они выполнялись каждый тик.
- JavaScript: большинство людей уже знают его, поэтому кривая обучения не очень крутая.
- Потокобезопасность: привязки JS к Java корректно обрабатывают параллелизм, вместо того чтобы заставлять вас спотыкаться о него.
- Знакомые шаблоны: стандартные JavaScript API работают так, как вы ожидаете
## Что вы можете построить

### Пользовательская автоматизация
Реагировать на события в игре:

```javascript
// Auto-respond to certain chat messages
on('chat', (event) => {
  if (event.message.includes('1v1?')) {
    sendChat('/duel accept');
  }
});
```
### Динамическая настройка модулей
Изменить настройки модуля в зависимости от состояния игры:

```javascript
// Increase reach when health is low
on('tick', () => {
  const health = player.getHealth();
  if (health < 8) {
    modules.get('Reach').setSetting('distance', 3.3);
  } else {
    modules.get('Reach').setSetting('distance', 3.1);
  }
});
```
### Пользовательские элементы интерфейса
Рисуйте собственные наложения:

```javascript
// Display a combo counter on screen
let combo = 0;
on('attackEntity', () => { combo++; });
on('damage', () => { combo = 0; });
on('render2d', (ctx) => {
  ctx.drawText(`Combo: ${combo}`, 10, 10, 0xFFFFFF);
});
```
### Профили для конкретного сервера
Автоматически переключайте конфигурации при подключении к разным серверам:

```javascript
on('serverJoin', (event) => {
  if (event.ip.includes('pvp.example.com')) {
    loadConfig('server-bypass');
  } else {
    loadConfig('ghost-default');
  }
});
```
