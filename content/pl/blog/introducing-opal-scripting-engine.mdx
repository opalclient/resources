Opal udostępnia silnik JavaScript GraalVM, pełne środowisko uruchomieniowe JS działające wewnątrz klienta Minecraft. Piszesz skrypty, które komunikują się z wewnętrznymi API Opal podczas trwającej rozgrywki. To nie jest rejestrator makr ani system przypisywania klawiszy. Jest to prawdziwe środowisko programistyczne z dostępem do systemu zdarzeń klienta, konfiguracji modułów i stanu gry.

<KeyTakeaways>

- Pełny środowisko uruchomieniowe JavaScript GraalVM, a nie rejestrator makr.
- Reaguj na zdarzenia w grze, zmieniaj ustawienia modułów i rysuj własny HUD.
- Skrypty ładują się w czasie działania, bez potrzeby ponownego uruchamiania, i działają w środowisku piaskownicy w odniesieniu do API Opal.
- Większość pozostałych klientów kończy się na podstawowych makrach.

</KeyTakeaways>

## Dlaczego GraalVM

GraalVM to wielojęzyczny środowisko uruchomieniowe od Oracle. Wybraliśmy je z konkretnych powodów:

- Szybkość: kompilacja JIT utrzymuje skrypty wystarczająco szybkie, aby działać w każdym cyklu.
- JavaScript: większość ludzi go już zna, więc krzywa uczenia się jest krótka.
- Bezpieczeństwo wątków: powiązania JS-Java prawidłowo obsługują współbieżność, zamiast zostawiać Cię do potknięcia się o nią.
- Znane wzorce: standardowe API JavaScript zachowują się tak, jak się tego oczekuje.

## Co możesz zbudować

### Niestandardowa automatyzacja

Reaguj na zdarzenia w grze:

```javascript
// Auto-respond to certain chat messages
on('chat', (event) => {
  if (event.message.includes('1v1?')) {
    sendChat('/duel accept');
  }
});
```

### Dynamiczna konfiguracja modułów

Zmieniaj ustawienia modułów w zależności od stanu gry:

```javascript
// Increase reach when health is low
on('tick', () => {
  const health = player.getHealth();
  if (health < 8) {
    modules.get('Reach').setSetting('distance', 3.3);
  } else {
    modules.get('Reach').setSetting('distance', 3.1);
  }
});
```

### Niestandardowe elementy HUD

Rysuj własne nakładki:

```javascript
// Display a combo counter on screen
let combo = 0;
on('attackEntity', () => { combo++; });
on('damage', () => { combo = 0; });
on('render2d', (ctx) => {
  ctx.drawText(`Combo: ${combo}`, 10, 10, 0xFFFFFF);
});
```

### Profile specyficzne dla serwera

Automatycznie przełączaj konfiguracje po dołączeniu do różnych serwerów:

```javascript
on('serverJoin', (event) => {
  if (event.ip.includes('pvp.example.com')) {
    loadConfig('server-bypass');
  } else {
    loadConfig('ghost-default');
  }
});
```
