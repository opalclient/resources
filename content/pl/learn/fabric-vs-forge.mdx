<KeyTakeaways>

- Fabric to lekki, szybko aktualizowany loader preferowany przez mody skupiające się na wydajności i użyteczności.
- Forge to starszy, cięższy loader z największym katalogiem dużych modów.
- Wybierz Fabric dla wsparcia wczesnych wersji i małego rozmiaru; wybierz Forge dla dużych pakietów modyfikacji.
- Mod jest skierowany do jednego z loaderów, więc dopasuj swoje mody do loadera, którego używasz.

</KeyTakeaways>

Fabric to lekki i szybki loader modów, który obsługuje nowe wersje Minecrafta w ciągu kilku dni i jest preferowany przez mody skupione na wydajności i użyteczności. Forge to starszy, cięższy loader z najszerszym katalogiem dużych modów dodających zawartość. Wybierz Fabric dla szybkości i wsparcia dla najnowszych wersji; wybierz Forge dla dużych pakietów modów i modów dodających zawartość, które są dostępne tylko tam.

## Co czym się charakteryzuje

Fabric to modularny loader modów, zbudowany wokół małego rdzenia oraz oddzielnie wersjonowanego [Fabric API](/pl/learn/what-is-the-fabric-api). Jest znany z szybkiego dostosowywania się do nowych wydań Minecrafta oraz czystego modelu łatania opartego na mixinach.

Forge to sprawdzony od lat loader, który napędzał większość dużych paczek modów przez lata. Zawiera obszerną własną powierzchnię API, co ułatwia tworzenie rozbudowanych modów, ale spowalnia aktualizacje loadera do każdej wersji.

## Jak się różnią

| | Fabric | Forge |
| :--- | :--- | :--- |
| Rozmiar | Lekki rdzeń + Fabric API | Duży, rozbudowane wbudowane API |
| Wsparcie dla nowych wersji | Dni po wydaniu | Tygodnie do miesięcy |
| Mocny w | Modach wydajnościowych i użytkowych | Dużych modach treściowych i paczkach modów |
| Model łatania | Mixiny | Coremods i własne haki |
| Kompatybilność między modami | API per-mod | Szerokie współdzielone API |

## Który wybrać

- Wybierz Fabric, jeśli chcesz wcześnie korzystać z najnowszej wersji Minecrafta, uruchamiać mody wydajnościowe lub chcesz minimalnego śladu.
- Wybierz Forge, jeśli Twoje ulubione mody treściowe lub paczki modów są tylko dla Forge, lub chcesz jedną dużą współdzieloną API do budowania.
- Wielu graczy ma oba, jeden profil na loader, ponieważ większość modów jest skierowana do jednego lub drugiego.

## FAQ

<FAQItem question="Czy Fabric jest szybszy od Forge?">
Sam loader jest lżejszy, a popularne mody wydajności, które z nim współpracują (Sodium, Lithium), działają na Fabricu. Surowa prędkość zależy od zainstalowanych modów, a nie tylko od loadera.

Bezpośrednio nie. Mod stworzony dla jednego loadera nie uruchomi się na drugim bez warstwy kompatybilności, dlatego sprawdź, dla jakiego loadera mod jest przeznaczony przed instalacją.

</FAQItem>
