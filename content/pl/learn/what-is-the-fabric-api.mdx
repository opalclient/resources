<KeyTakeaways>

- Fabric API to współdzielona biblioteka haków i narzędzi, względem której tworzonych jest większość modyfikacji Fabric.
- Jest ona celowo oddzielona od ładowarki: mały ładowarka szybko się aktualizuje, API ewoluuje niezależnie.
- Zapewnia wydarzenia, networking, rejestry i narzędzia do renderowania, na których polegają liczne modyfikacje.
- Brak lub przestarzała wersja Fabric API jest najczęstszą przyczyną awarii modyfikacji podczas uruchamiania.

</KeyTakeaways>

API Fabric to współdzielona biblioteka haków i pomocników, względem której tworzonych jest większość modyfikacji [Fabric](https://fabricmc.net/). Rdzeń ładowarki Fabric jest celowo mały, dlatego właśnie API wypełnia luki: zdarzenia, sieci, pomocnicy rejestru, narzędzia renderowania. Jeśli modyfikacja Fabric ulega awarii podczas uruchamiania z powodu błędu braku klasy, przyczyną jest zazwyczaj brak lub przestarzała API Fabric.

</Callout>

## Dlaczego jest oddzielone od ładowacza

Fabric celowo dzieli system na dwie części. Ładowacz pozostaje mały i stabilny, a API Fabric zawiera większy, szybszy zestaw współdzielonych interfejsów. To rozdzielenie pozwala ładowaczowi szybko aktualizować się do nowej wersji Minecrafta, podczas gdy API rozwija się we własnym tempie.

## Co zapewnia

API daje modom wspólny, stabilny sposób robienia rzeczy, które są potrzebne wielu modom:

- Cykl życia i zdarzenia rozgrywki, do których można się podłączyć.
- Pomocniki sieciowe dla niestandardowych pakietów.
- Pomocniki rejestru i zawartości do dodawania przedmiotów, bloków i bytów.
- Pomocniki renderowania i zasobów współdzielone przez mody.

Bez niego każdy mod musiałby na nowo wymyślać te elementy i psuć sobie nawzajem działanie.

## Czy go potrzebujesz?

Prawie zawsze, tak. Większość modów Fabric wymienia API Fabric jako zależność, więc instalujesz je raz jako mod wraz z innymi. Dopasuj jego wersję do swojej wersji Minecrafta.

## FAQ

<FAQList>

<FAQItem question="Czy wymagana jest Fabric API?">
Zazwyczaj tak. Mod, który od niej zależy, nie załaduje się bez niej. Kilka samodzielnych modów jej nie potrzebuje, ale zainstalowanie jej jest bezpiecznym domyślnym rozwiązaniem.
</FAQItem>

<FAQItem question="Gdzie to zdobyć?">
Z oficjalnych kanałów Fabric oraz u popularnych twórców modyfikacji. Zawsze wybieraj wersję, która odpowiada Twojej wersji Minecraft.
</FAQItem>

<FAQItem question="Dlaczego nie jest wbudowane w ładowacz?">
Pozwala to zachować niewielki rozmiar ładowacza i umożliwia szybkie aktualizacje, podczas gdy współdzielona API ewoluuje oddzielnie. To celowa decyzja projektowa w Fabricu.
</FAQItem>
